# 短剧剧本创作完全指南

## 核心理念

**短剧定位：** 每集1-1.5分钟，部分60-100分钟共40-100集的短视频连续剧

### 短剧写作三要素

1. **噱头设置** - 博人眼球、引人关注、调动情绪
2. **结构框架** - 立好人设→定好首尾→找到噱头→开篇切入→往首尾的中间填剧情
3. **核心技巧**
   - 台词要有力度，时间线要紧凑
   - 矛盾前置：开篇即矛盾，无需过多铺垫
   - 每集必须具备：矛盾点、爆发点、爽点、结尾钩子
   - 制造紧张感，拉扯情绪，增强代入感

### 情绪爆发原则

- 台词饱满、有爆发力、疯狂
- 人物情绪要外化：绿茶光明正大地作恶、反派彻底之恶、配角张牙舞爪
- 情绪拉扯 > 剧情铺垫

## 第一步：受众痛点分析

### 痛点挖掘方法

以女频为例，常见痛点：
- **典型痛点:** 老公在生孩子时漠不关心
- **加工方法:** 将小事夸张化、扭曲化，制造对比
- **示例:** 
  - 女主为了生孩子几近死亡
  - 生完孩子发现男主在陪女配产检，有说有笑
  - 这同时产生**对比冲突**：男主对女主的态度 VS 对女配的态度 → 冲突增强
### 剧情创新原则

同样的设定，用不同的表现方式呈现 → 这就是创新

## 第二步：剧集框架设计（集纲写法）

### 三段式结构

1. **起因（篇幅10%）**
   - 以引发强烈冲突的动作开场
   - 示例：擦边、扇耳光、将人从山上扔出去、绑架等
   - 目的：快速吸引观众

2. **经过（篇幅80%）**
   - 核心：羞辱、压迫主角
   - 通过持续冲突增强观众对主角反击的期待
   - 示例：女主嘲讽男主没钱、没出息
   - 保持台词紧凑、围绕当前冲突

3. **结果（篇幅10%）**
   - 留下悬念钩子
   - 示例：陷害被他人质疑、秘密被发现、身份要揭晓
   - 让观众对后续剧情充满期待

### 台词设计原则

**每一句台词都要围绕当前冲突展开**
- 只讲必要信息，删除无关枝节
- 例外：只有背景信息对理解动机很关键时才保留
- 能略写就略写，能弱化就弱化

## 第三步：快速写出好剧本的四大法则

### 法则1：极速节奏

- **开篇设计**
  - 呈现主题
  - 快速调动情绪
  - 爽点充足
  - 钩子吸引人

- **台词要求**
  - 短而精
  - 爆发力强
  - 拒绝冗长

### 法则2：爽感最大化

**何为爽感？** 观众期待的满足感

常见爽感类型：
- 暴富（获得财富）
- 获得特异功能（超能力觉醒）
- 重生改命（穿越、时间重启）
- 逆袭翻盘（身份翻转、地位逆转）
- 身份悬殊下的非凡成就（穷小子征服女神、获得豪门认可）

### 法则3：微创新

- 人设创新（反差设定）
- 反套路创新（打破常规预期）

### 法则4：画面感 + 强冲突

- 保持鲜明的视觉冲击
- 矛盾冲突需要强烈、尖锐
- 避免平淡叙述

## 第四步：剧本大纲创作

### 核心原则
**大纲是事件的罗列，而非描述性文字**

### 创作禁忌（必须规避）
| 禁忌 | 后果 | 解决方案 |
|------|------|----------|
| 主线不明确，情节松散 | 观众抓不住故事 | 聚焦一条清晰主线 |
| 节奏慢，缺乏冲突和吸睛点 | 用户流失 | 每集必有冲突+爽点 |
| 情节同质化严重 | 审美疲劳 | 创意突破，反套路 |
| 配角过多，人物混乱 | 叙事混乱 | 控制主要人物数量 |
| 体量不足 | 无法撑起100集 | 前期充分铺垫矛盾 |
| 逻辑混乱，后期圆不回 | 观众质疑 | 前期规划清晰的逻辑链 |
| 铺垫过多，切入太慢 | 流失早期观众 | 开篇3秒进入冲突 |
| 缺少人物小传 | 人物行为生硬 | 先写人设后写剧本 |
| 背景介绍过长 | 拖沓无聊 | 用对话自然穿插背景 |
| 忽视拍摄成本 | 无法落地制作 | 考虑实际拍摄可行性 |

## 第五步：短剧写作技巧体系

### 短剧的本质特征

**时长规格**
- 每集1-1.5分钟
- 总时长60-100分钟（40-100集）

**写作形式**
- 故事模型：谁→在什么场所→发生了什么事
- 表现形式：以对话为主的对话文
- 与小说的区别：注重情绪 > 注重文笔

### 创作规则

#### 规则1：情绪优于文笔

- **情绪要求**
  - 强烈、直接、有时粗俗
  - 拒绝含蓄和精致
  - 人物情绪要外化、极端

- **人物设定**
  - 单一、扁平化（代表一种情绪类型）
  - 少见多怪、大喊大叫、不通情理皆可

#### 规则2：情绪价值最大化

**前期阶段**
- 充分体现虐、恶心等负面情绪
- 情绪拉扯强度拉满
- 积累观众对主角反击的期待

**后期阶段**
- 能洗白人物就洗白（给予救赎）
- 洗不白则深化虐线（女主自杀或死不原谅、男主自杀等虐文向）

#### 规则3：反派人物的情绪表达

**台词粗俗 ≠ 质量低 → 是极端情绪的表现**
- 说话越粗俗难听越好
- 目的：表现极端、尖锐的情绪冲突

### 剧本基本格式

**标准格式**
```
场次-1（第N场第1个剧情） | 地点 | 内/外景 | 日/夜 | 出场人物
△ [叙述：除对话外的动作/环境信息]
人物名：[对话]
△ [叙述]
人物名：[对话]
```

**格式说明**
- `△` 符号：表示叙述部分（动作、环境、心理）
- 一集构成：2-3个场景，400-500字
- 重点：少环境描写，多对话和动作
## 第六步：剧本大纲构建（100集体系）

### 大纲的四大必要元素

#### 1️⃣ 人物小传（基础）

每个主要角色需明确：
- 基本信息：年龄、性格、气质
- 背景故事：是否有重大变故导致黑化
- 人物关系：与男女主的具体关系
- 角色结局：预设这个角色的最终走向

#### 2️⃣ 大背景（故事源头）

定义：男女主发生一系列纠葛的原因

**重要但非必显**
- 写正文时不需强行解释
- 在人物对话中自然带出
- 示例：《厉总，夫人她罪不至死》
  - 背景：女配需要献血 → 男主求女主献血 → 女主提条件"娶我" → 冲突起源

#### 3️⃣ 主线（全剧内核）

定义：全剧的故事关键词、主角的终极任务

**特点**
- 所有剧情都围绕主线展开
- 支线可多，但必须服从主线
- 示例：《真假千金》
  - 主线：找到失踪的父亲
  - 所有100集剧情都在推进这个目标

#### 4️⃣ 大纲内容规划

**基本原则**
- 按照**起承转合**的节奏制定
- 一句话说清一件事（信息密度高）
- 逻辑通顺：每个剧情间的因果关系要清晰
- 禁止突兀：想什么写什么 → 观众看不懂

### 100集大纲的分阶段写法

**推荐方案**
```
第1阶段：1-20集（前期铺垫）
 → 故事梗概 + 人物介绍 + 冲突设置

第2阶段：20-40集（故事展开）
 → 矛盾升级 + 配角登场 + 反转埋点

第3阶段：40-60集（高潮前奏）
 → 情势恶化 + 跌入深渊 + 准备反击

第4阶段：60-80集（大高潮）
 → 最终对抗 + 惊天反转 + 反派制裁

第5阶段：80-100集（收尾）
 → 人物救赎 + 情感落定 + 结局圆满
```

### 大纲6大核心要点

| 要点 | 说明 | 示例 |
|------|------|------|
| **先定首尾** | 从结局倒推前期设定 | 已知结局是反派被制裁，所以前期要充分积攒仇恨 |
| **节奏递进** | 矛盾+反转 → 情绪持续升级 | 每5集一次反转，收视曲线呈上升 |
| **前30集标准** | 每集必有冲突+打脸+钩子 | 保证用户不弃坑 |
| **钩子设置** | 激发观众继续观看的动力 | 9-15集是付费转折点 |
| **背景融入** | 用对话自然展现，避免闪回 | 人物聊天间提及过去恩怨 |
| **直接表达** | 忌讳含蓄、委婉、拐弯抹角 | 删除所有"话中有话"的台词 |

### 大纲内容示范

**错误示范**
```
第1集：女主可怜兮兮地在雨中走着，心里想着自己的处境...
（太多心理描写，无实际事件）
```

**正确示范**
```
第1集：女主被渣男当众甩耳光，并嘲讽"你配不上我"，
全公司员工都笑了，女主哭着跑出办公室。
（清晰的事件、冲突、结果）
```
## 第七步：核心三要素详解

### 要素1：矛盾冲突（故事心脏）🔥

**冲突等级标准**
- ⭐ 冲突要尖锐、震撼
- ⭐ 具有爽感、揪心感、视觉精彩
- ⭐ 能激发情绪反应

**冲突来源在哪？（多元冲突体系）**

1. 男主 ↔️ 女主（感情冲突）
2. 男主 ↔️ 女配（权力/感情冲突）
3. 女主 ↔️ 女配（对立冲突）
4. 女主 ↔️ 男配（利益冲突）
5. 男主 ↔️ 男配（身份冲突）
6. 主角 ↔️ 命运（内心冲突）

**设计原则**
- 多方面制造冲突，避免单一对立
- 每个冲突都要服从主线
- 冲突升级递进，不能重复

---

### 要素2：台词设计（情绪承载）💬

**台词黄金法则**

| 原则 | 做法 | 反面 |
|------|------|------|
| 体现人物性格 | 每个角色的说话方式都不同 | 所有人说话都一样 |
| 声线一致性 | 霸总说话要霸、草包说话要蠢 | 人物声线混乱 |
| 情绪露骨 | 直白表达情绪 | 含蓄、拐弯抹角 |
| 信息凝聚 | 一句话只传递一个信息 | 台词冗长、信息超载 |

**台词示范**
```
错误：「其实我也很难受，因为我爱你，但是最近这些事情让我觉得...」
（冗长、模糊、听不清情绪）

正确：「我恨死你了！" 或 "我离不开你！"
（短、直、情绪清晰）
```

---

### 要素3：舞台设计与场景说明 🎬

**舞台说明的完整要素**

包括：人物、时间、地点、服装、道具、布景等情景信息

**剧本的完整架构**
```
开端 → 设置矛盾 → 冲突升级 → 再设矛盾 → ... → 高潮 → 结局
```

**这不是线性的！而是反复的矛盾-解决-新矛盾的循环。**

**主题明确性**
- 这个剧讲什么故事？必须清晰
- 主题要贯彻全剧（不能中途变主题）

**创造冲突的维度**
- 角色与角色的对抗
- 角色与自身价值观的对抗
- 角色与外界环境的对抗

**转折点设置**
- **位置**：故事前端和后端
- **作用**：制造意外、改变观众预期
- **前段转折**：开启故事、展示主角抉择
- **后段转折**：反转局面、绝地反击

**对白标准**
- 精简：删除所有废话
- 每句台词都要推动剧情或表现情绪
- 宁缺勿滥：少说 > 说得多

## 大纲模板（参考格式）

### 完整大纲框架

```
【故事梗概】（100-150字）
用最简洁的语言说明故事是什么

【主线】（200字）
全剧的核心任务和推动力

【支线】（200字）
辅助性的情节线索（最多2-3条）

【故事发展】（100字）
阶段性的情节进展

【故事高潮】（100字）
最大的对抗和情绪爆发

【故事转折】（100字）
局势反转、结局突变的关键点

【彩蛋】（50-100字）（可选）
结局之外的惊喜

【集纲】
一句话描述每集核心事件
示例：第1集，女主被甩耳光，进入绝望阶段

【按集数标注】
范例：第1-8集｜故事展开阶段｜主要情节为女主遭遇背叛，陷入困顿
```


19:聚会/宴会时，遭到会场人员的歧视、羞辱，路过的大人物却纷纷对主角点头哈腰
20、其他人害怕、闻风表胆的人物，却是主角看不起的手下败将
21、能够掌控世界的大人物，实际上是主角的傀儡

25、某些事在别人看来无法解决，是惊天大难题，而对主角来说却是小菜一碟

短剧的8种剧情写法
1、侮辱和背叛：被男友、闺密、亲近的人侮辱、背叛
2、误会：被男主无脑误会
3、暴力对待：身体暴力、情感暴力
4、肢体冲突、情感冲突：虐身虐心
5、言语攻击：配角对主角的言语攻击
6、不信守承诺：人物出尔反尔
7、童年阴影：剧本中人物的某种行为，是童年遭受了某种阴影而导致的
8、欺诈：观众有上帝视角，知道人物被期诈，而剧中人物却不知道自己被欺诈

一、强拆CP

4、分裂主角和其他角色的友情，让主角孤立无援

1、装可怜，引起主角同情心

7、散布谣言，破坏主角之间的信任
8、在主角最需要帮助的时候，故意失联

4、四处散播谣言，让主角声誉扫地

6、假扮主角的敌人，破坏主角的计划
四、利用算计
1、利用主角的仁慈之心，诱导主角犯错

3、利用主角的弱点，从内心打击主角的信心和意志
4、利用主角的对手，间接破坏主角的计划
5、利用主角的好心，让主角走向灭亡
6、利用主角的家世，制造负面效应
7、利用主角的弱点，让主角陷入困境
8、利用主角的心理，暗中操控主角的行动
9、利用主角的缺陷，毁掉主角名声

4、在主角最需要帮助的时候，制造麻烦

6、利用主角的仇恨，制造麻烦

剧本台词，是角色传达信息，引发矛盾冲突的关键点
1、台词要口语化：简短有力，言简意赅
2、台词要体现个人特色，有自己的属性，比如霸总就要有霸总的样子，能命令的时候绝不请求，多观察生活里的人的说话方式，总结什么性格的人会说什么样的话
3、好的台词，一定是在戏剧情景里，配合人物动作产生的，起推动剧情的作用，把对话放进剧情里，用事件表现，简短有力

剧本钩子：留悬念吸引观众看下一集，在冲突发生时卡点结束
1、留悬念：比如抓到反派时突然卡点，或者发现了蛛丝马迹突然卡点
2、人物表情：比如突然之间人物露出不可思议的表情，或者害怕，疑惑的表情，然后卡点，让人想一探究竟
3、谈话：聊到关键部分突然停止
4、突然喊话：比如住手、停手、松手、然后卡点

短剧的开头和前十集怎么写
开头：三秒内吸引观众
- 情绪点：大哭、紧张、冲突爆发、喊话、求救、怒吼
- 视觉点：车祸、扇耳光、羞辱、香艳镜头、掀桌子、绑架、豪车场面
- 开头要同时给出“谁”“要做什么”“为什么不行”，并在最后留下悬念

前三集：建立关系、塑造冲突、埋下反转线
- 说明人物关系、出场动机、事件起因
- 设定小高潮，迅速拉动情绪
- 结尾留钩子：反转、发现、突发危险、关键情报

第4集：揭示关键信息或大事件
- 让主角开始反击或调整策略
- 让反派出现更强硬的面，推动局势升级

第5-8集：多线压迫，情绪递进
- 反派升级、误会加深、支线人物加入冲突
- 主角受到多方打压，优势被拆解，失控感增强
- 让观众不断担心主角能否撑住

第9-10集：卡点集，留足悬念
- 危机爆发、秘密露出、反转前兆、爽点预热
- 尾声不必完全解决，留出下一阶段的节奏

强悬疑点
- 电话来电、神秘访客、突发消息、表情凝重
- 台词："不好了"、"坏了"、"他回来了"、"你来晚了"
- 让观众在结局处“想知道接下来怎么救/怎么反击”

剧情创新
- 多看同类小说、短剧，积累爆款梗
- 创新不在于凭空发明，而在于组合、反转、拆解已有套路
- 优先选择市场验证过的题材，改编成功率更高
- 找到一个与主角身份、关系、目标相关的“新鲜点”作为切入

剧情平淡怎么破
- 找到施力点与受力点：施力要狠，受力要大
- 让冲突直接落在主角或者与主角高度相关的人身上
- 避免“旁观者式冲突”或“无关痛痒”的事件
- 每一集都要有明确的矛盾节点，避免情节过于平铺

台词的作用
- 推进剧情、压情绪、展现背景信息
- 每句话只抛出一个重点信息
- 台词要口语化、符合人物性格，配合动作和场景
- 避免为了交代而说教式台词

短剧常用计谋
1、反间计：反派以为掌控全局，结果主角早有准备
2、美人计：反派用美色引诱，主角反杀反转
3、借尸还魂：主角假死后以全新身份回归
4、胜追穷寇：追击落水狗，彻底打垮反派
5、大器晚成：反派外强中干，最强位置始终得不到
6、计中有计：表面上的诱导背后有更深布局
7、得道多助：主角身边人才辈出，势力不断扩张
8、关门捉贼：反派进入圈套后被瞬间反制
9、偷梁换柱：反派以为得手，实际被掉包
10、狐假虎威：反派借势张扬，结果为主角所控
11、苦肉计：为了赢取信任，故意示弱或受伤
12、指桑骂槐：表面批评他人，实则攻击目标
13、瞒天过海：利用身边人掩护真实目的
14、添油加醋：反派捏造事实，借紧张局势加码
15、反客为主：反派原本炫耀，结果被主角抢回主导权
16、招摇撞骗：反派借伪装伪造证据，自食其果
17、黄袍加身：反派夺走主角一切，最终被反噬
18、落井下石：借主角危机伺机袭击
19、借题发挥：利用无关事件置主角于死地
20、趁火打劫：反派趁主角受伤或被打压时追加攻击
21、兔死狗烹：同盟为利益互相残杀
22、步步为营：主角逐步蚕食反派势力
23、以毒攻毒：用反派的狠辣回击反派
24、调虎离山：引主角离开，攻其所爱
25、礼尚往来：以彼之道还治彼身
26、顺水推舟：借反派之力铺路
27、以暴制暴：用暴力回应暴力
28、连环计：一层不够，再设多层反击
29、屈打成招：通过压迫逼对方就范

冲突设定
1、冲突要新颖、有爽点
2、先想爽点，再设计冲突
   - 例如：主角获得金手指，前期冲突可以来源于“穷”“被欺负”“身世被揭露”
   - 女频爽点常围绕感情展开，例如误会虐恋、绝症、豪门身份、身份之子
3、冲突应发生在主角身上
   - 与主角无关的危机无法带来紧张感
   - 强行让主角参与会让剧情显得莫名其妙
   - 开头部分冲突首选主角本人，其次是与主角高度相关的人

写正文的核心思路
1、确定本集中心事件（要精彩、能拉扯情绪）
2、撰写人物小传，明确角色关系、动机和性格
3、台词信息量适中，一句话尽量只传递一个信息
4、避免正文太散、矛盾不集中、人物行为生硬
