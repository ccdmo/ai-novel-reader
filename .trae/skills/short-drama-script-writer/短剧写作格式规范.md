## 第二部分：场景标准格式

### 2.1 场景头标准写法

**标准格式**
```
【场景 X】| 地点名称 | 内景/外景 | 时间 | 人物
```

**补充说明**
- 📍 **地点名称**：明确、具体（不要写"办公室"，要写"总裁办公室33楼") 
- 🏛️ **内景/外景**：简写为【内】或【外】
- ⏰ **时间**：【日外】【夜内】【黎明】【傍晚】等（粗拆即可）
- 👥 **人物**：这场出现的所有说话人，用"/"分隔

**四个范例**

```
【场景1】| 总裁办公室 | 【内】【日】| 女主、总裁、秘书

【场景2】| 办公大厅 | 【内】【日】| 女主、渣男上司、同事们

【场景3】| 停车场 | 【外】【傍晚】| 总裁、女主

【场景4】| 酒吧包间 | 【内】【夜】| 渣男、神秘人
```

### 2.2 场景内部元素标准

```
【场景 X】| 【地点】| 【内/外】【日/夜】| 【人物】

△ [环境/动作叙述]

【人物A】：
[台词]

△ [动作/表情叙述]

【人物B】：
[台词]

△ [转场/结尾叙述]
```

**符号说明**
- `△` = 叙述符号（环境描写、动作描写、表情心理）
- 【】= 标注符号（清晰结构）
- 【人物】: = 说话人符号

## 第三部分：台词写作规范

### 3.1 台词的标准形式

```
【人物名字】：
[台词内容]
```

**关键规范**

✅ **必须做**
- 人物名字后加英文冒号 `：`
- 台词单独占一行
- 台词要短而精，拒绝冗长
- 每句话围绕当前冲突

❌ **禁止做**
- 人物名写在台词同一行（混淆格式）
- 背景信息过多（应该用△叙述）
- 含蓄委婉的话（直白表达）
- 多余的心理活动

### 3.2 台词的情绪标注

**方式1：括号标注**
```
【女主】：
（哭腔、委屈）你怎么能这样对我！
```

**方式2：行动标注**
```
【女主】：
（转身、眼泪）你怎么能这样对我！
```

**方式3：混合标注**
```
【女主】：
（尖叫、转身击打沙发、眼睛通红）你怎么能这样对我！
```

**常用情绪标注**
- （哭腔、委屈、抽泣）
- （怒火中烧、尖叫、拍桌子）
- （冷笑、嘲讽、转身离开）
- （惊恐、往后退、颤抖）
- （甜蜜、害羞、低眉顺眼）
- （阴狠、冷淡、眼神刀锋）

### 3.3 台词长度控制

**打分标准**

⭐⭐⭐⭐⭐ 完美台词（10-30字）
```
【总裁】：
（冷冷）你试试看，再敢让人欺负我的妻子。
```

⭐⭐⭐⭐ 良好台词（30-60字）
```
【总裁】：
（上前、掐住渣男的脖子）怎么，觉得自己很能耐？
敢在我的地盘欺负我的女人，我看你是活腻歪了。
```

⭐⭐⭐ 可接受（60-100字）
```
【总裁】：
（居高临下、冷漠）我能忍你很久了，从你第一次骚扰她开始，
我就一直在看着。本来想给你个机会改过自新，
但现在看来你是真的不懂什么叫知道错了。
```

⭐⭐ 需要调整（100-150字，开始显得冗长）
❌ **禁止**（150字以上，完全违反短剧规范）

### 3.4 对话节奏规则

**规则1：快速对话**（紧张冲突）
```
【女主】：你为什么要这样骗我！

【男主】：我没有骗你。

【女主】：没有？那你怎么解释？

【男主】：（转身）没什么好解释的。

【女主】：（跑上前、拉住他）你必须解释！
```

**规则2：单人独白**（显示心理）
```
【女主】：
（看着窗外、喃喃自语）
他到底是真心还是假心...
我真的分不清了。
```

**规则3：重叠对话**（表现情绪激烈）
```
【女主】：
你这个骗子！

【男主】：
听我解释——

【女主】：
没什么好解释的！
```

## 第四部分：动作和表情描写

### 4.1 △叙述的标准形式

```
△ [动作/表情/场景描写]
```

**叙述的内容分类**

| 分类 | 示例 | 字数 |
|------|------|------|
| **动作** | △ 女主转身、走向门口 | 5-15字 |
| **表情** | △ 总裁面无表情地看着她，眼神里闪过冷意 | 15-25字 |
| **环境** | △ 办公室的窗户外，整个城市灯火通明 | 15-25字 |
| **混合** | △ 女主转身、眼泪滑过脸颊、跑出办公室 | 20-30字 |

### 4.2 动作描写的精简规则

**原则**
- 🎯 围绕冲突服务（不要无关的美化）
- ⏱️ 短而劲（10-30字最佳）
- 🎬 表现力强（动词要有活力）

**错误示范及修正**

❌ **过度美化/文艺**
```
△ 女主美丽的面容上闪过一丝悲伤的泪水，
  在灯光下显得晶莹剔透，如同珍珠般润泽...
```

✅ **修正后**
```
△ 女主眼泪滑过脸颊，转身逃离
```

---

❌ **冗长废话**
```
△ 总裁从座位上缓缓站起来，整理了一下袖口，
  走到窗边，看着窗外的城市，思考着人生...
```

✅ **修正后**
```
△ 总裁起身、走到窗边、久久不语
```

---

❌ **无关细节**
```
△ 女主穿着蓝色裙子，头发飘飘然，画着精致的眼妆，
  走进了办公室...
```

✅ **修正后**
```
△ 女主走进办公室
```
（除非蓝色裙子/眼妆对冲突有直接影响）

### 4.3 表情和心理的写法

**有效表情描写**
```
△ 总裁的眼神瞬间冷下来，握紧拳头

△ 女主低下头，肩膀微微颤抖

△ 渣男脸色惨白，往后退了一步

△ 秘书惊讶地瞪大眼睛，嘴巴成"O"形
```

**无效表情描写**
```
❌ 总裁生气了 → 修正：△ 总裁眼神冷了两个度

❌ 女主难过 → 修正：△ 女主眼泪模糊了双眼

❌ 他们很尴尬 → 修正：△ 两人面面相觑、沉默
```

## 第五部分：完整场景范例

### 5.1 范例1：爽感场景（高能打脸）

```
【场景2】| 办公大厅 | 【内】【日】| 女主、渣男上司、同事们

△ 女主低着头、来回整理文件
  渣男上司走近、目光轻佻

【渣男上司】：
（靠近、伸手想摸女主的脸）
小美啊，今晚有没有时间，咱们一起去——

△ 突然，电梯门打开
  总裁阔步走出，眼神一片冰冷

【总裁】：
（上前、一把扒开渣男的手、护在女主身前）
你的手是想断了吗？

【渣男上司】：
（吓得往后退）
总...总裁！

【总裁】：
（上前、指着渣男、声音冷到极致）
这是我的妻子。再敢动她一根手指，
我会让你这辈子都下不了床。

△ 办公大厅瞬间安静
  所有同事都瞪大眼睛

【同事甲】：
（尖叫）什么？！妻子？！

【同事乙】：
（转身、窃窃私语）我就说他们两个不对劲！

△ 总裁转身、拉起女主的手、走向电梯
  众人目光追随

【总裁】：
（回头、冷冷扫过渣男一眼）
滚出这家公司。

△ 电梯门关闭
  渣男脸色死灰，同事在身后指指点点
```

**分析**
- ✅ 冲突浓度高（3秒内打脸）
- ✅ 爽感足（总裁霸气宣布+渣男下跪）
- ✅ 节奏快（动作频繁、台词简洁）
- ✅ 字数合理（约380字）

### 5.2 范例2：情感场景（拉扯感情线）

```
【场景3】| 总裁办公室 | 【内】【夜】| 女主、总裁

△ 女主坐在沙发上、低着头
  总裁靠在办公桌边、看着她

【女主】：
（不敢抬头）
你...为什么要这样做？

【总裁】：
（沉默片刻、走近）
为什么？

△ 女主抬起头、眼眶微红

【女主】：
（哭腔）
我们...才认识一天啊。

【总裁】：
（蹲下、抬起她的下巴、强势）
不管认识多久，我不会允许任何人伤害你。
这是我的原则。

△ 女主眼泪滑落
  总裁伸手擦去、眼神柔和了一丝

【女主】：
（颤抖）你这样...会让我分不清现实。

【总裁】：
（起身、转身走向窗边）
那就不用分清。

△ 女主呆呆地看着他的背影
  镜头拉远...

△ 电话铃声响起（转场暗示）
```

**分析**
- ✅ 情感线拉扯（推拉张力）
- ✅ 速度适中（有铺垫、有反转）
- ✅ 悬念留下（"分不清现实"+ 电话铃声）
- ✅ 字数合理（约320字）

### 5.3 范例3：反转场景（埋伏笔）

```
【场景4】| 停车场 | 【外】【傍晚】| 渣男、神秘人

△ 渣男走向自己的车、神情沮丧
  一个身影从暗处走出

【神秘人】：
（声音压低、带着口音）
你刚才的表现很狼狈啊。

【渣男】：
（转身、惊恐）
你...你是谁？

【神秘人】：
（掏出手机、展示合照）
看着，这就是总裁的妻子。
你觉得，要不要把这件事...
让某些人知道呢？

△ 特写：手机屏幕上的合照
  总裁和女主亲密地靠在一起

【渣男】：
（眼神变了、贪心）
你...想要什么？

【神秘人】：
（冷笑）
很简单。帮我打探一些信息。

△ 镜头拉远
  两人的身影消失在停车场的阴影里
```

**分析**
- ✅ 悬念制造（神秘人身份未知）
- ✅ 反转埋伏（渣男从被害者变共谋者）
- ✅ 为后续续剧开路（新的冲突源）
- ✅ 字数合理（约250字）

## 第八部分：常见错误纠正

### 常见错误1：场景标注不清

❌ **错误**
```

```

✅ **正确**
```
【场景1】| 办公大厅 | 【内】【日】| 女主、渣男上司、总裁
```

---

### 常见错误2：台词过长

❌ **错误**
```
【总裁】：
你听我说，从我第一次见到你的那一刻起，
我就知道你不是一个普通的女孩。
你身上有一种光芒，吸引着我。
所以我今天做出了这个决定，
就是要在众人面前宣布你是我的妻子...
```

✅ **正确**
```
【总裁】：
（走近、强势）
这位是我的妻子。
再敢碰她一根手指，我会让你后悔。
```

### 常见错误3：场景内容冗余

❌ **错误**

  照在大理石地板上闪闪发光。

```
△ 女主低着头、不安地整理文件
```

## 第九部分：检查清单

### 剧本完成前的自检项

```
【格式检查】
□ 集数、标题、字数、时长都已填写
□ 场景标注完整（地点/内外景/时间/人物）
□ 人物名字后有英文冒号
□ △符号用于所有叙述部分
□ 台词用【】标注，清晰可辨

【内容检查】
□ 开篇有冲突（1-2句内进入矛盾）
□ 中间段落持续拉扯情绪
□ 每句台词都围绕当前冲突
□ 结尾留下悬念钩子
□ 本集具备四要素（矛盾点/爆发点/爽点/钩子）

【字数检查】
□ 总字数在400-500字范围内（允许±50字）
□ 场景个数在2-3个之间
□ 没有冗长段落（单个叙述/台词超过50字）

【情绪检查】
□ 所有关键台词都有情绪标注（括号内容）
□ 动作描写体现了人物情绪状态
□ 冲突的情感强度逐步升级

【逻辑检查】
□ 场景转换有逻辑（不能跳跃太大）
□ 人物出场/退场交代清楚
□ 悬念设置合理（能在下一集有所回应）
```
