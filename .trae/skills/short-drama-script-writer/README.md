## 🚀 快速开始

### 针对不同用户的推荐流程

**场景1：我要快速生成一个短剧框架**
```
1. 读："短剧剧本核心分析.md"的"四大成功模式"部分
2. 选择最适合的模式（爽文/甜宠/权谋/穿越）
3. 参考格式规范.md中的框架生成部分
```

**场景2：我要创作一集具体的剧本**
```
1. 读："短剧写作格式规范.md"的前五部分
2. 参考其中的完整范本
3. 按照格式标准开始写作
```

**场景3：我要检查剧本质量**
```
1. 读："短剧剧本创作提示词.md"的检查清单
2. 使用40项检查清单评估自己的剧本
```

**场景4：我想了解爆款剧本的规律**
```
1. 读："短剧剧本核心分析.md"全文
2. 理解四大模式和通用规律
3. 应用这些规律到自己的创作中
```

## 📊 核心概念速查表

### 三段式结构
```
起因(10%) → 经过(80%) → 结果(10%)
```

### 每集必备四要素
- ✅ 矛盾点 - 明确的冲突对立
- ✅ 爆发点 - 情绪或剧情的高潮
- ✅ 爽点 - 观众期待的满足感
- ✅ 钩子 - 吸引看下一集的悬念

### 四大成功模式
| 模式 | 对标作品 | 爽点特点 | 成功率 |
|------|---------|---------|--------|
| 男主爽文 | 《神豪归来》| 打脸+掌控 | 🥇 最高 |
| 古装权谋 | 《深宫逆袭》| 权力+智斗 | 🥈 很高 |
| 女主甜宠 | 《别惹总裁》| 宠妻+虐渣 | 🥉 高 |
| 穿越混搭 | 《战神御厨》| 双重快感 | 🎖️ 中高 |

## 💰 关键数据指标

### 标准剧本参数
- 每集时长：1-1.5分钟
- 每集字数：400-500字
- 每集场景数：2-3个
- 总集数：100集
- 冷启动要求：3秒进入冲突

### 付费转折点
- 9-15集触发第一次付费意愿
- 需要重大反转或秘密暴露

## 🎯 常见问题

**Q: 这个Skill能完全替代人工编剧吗？**
A: 不能。Skill提供框架和优化建议，但最好的剧本需要人工的情感投入和创意突破。

**Q: 生成的剧本是否保证能成为爆款？**
A: 不能保证。Skill基于200部已验证剧本的规律，但最终成功还取决于演员、导演、宣发等多方面因素。

**Q: 可以用于其他类型的短剧吗？**
A: 可以。虽然优化了四种主流类型，但基础的格式规范和节奏控制适用于所有短剧。
