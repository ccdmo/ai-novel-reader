# 玄门神算卦太灵，火爆全京城！

## 第1集

【视觉钩子】
△青砖黛瓦的萧府大门外，一辆破旧马车缓缓停下。车帘掀开，一袭素白衣裙的清瘦少女踏下马车，微风拂起她额前碎发，露出精致却略显苍白的面容。身后只有一个背着破旧包袱的小丫鬟。

【核心冲突】
△姜氏立于台阶之上，看见萧云灼的第一眼便皱紧了眉，冷声道："站在那儿别动！先把自己洗干净再进门，别把外面的晦气带进萧家。"

【爽点释放】
△萧云灼不急不缓地抬眸，唇角微微勾起，懒洋洋道："母亲如今成了下凡的天仙？竟还管得这般宽，管到女儿身上了。"

【情感拉扯】
（萧云灼内心os：十二年了，姜氏，你这副嘴脸倒是一点没变。）她不动声色地扫了一眼姜氏的面相，眼底闪过一丝冷意。

【悬念钩子】
→ 姜氏被噎住，脸色铁青，正欲发作，却被身旁的婆子拉住。萧云灼的目光越过众人，看向府内某个方向——那里，正有一缕淡淡的黑气缭绕。
→ 破旧包袱里，似乎有什么东西微微动了动。

## 第2集

【视觉钩子】
△萧云灼被管家领着七拐八绕，来到府中最偏僻角落的一处小院。院墙斑驳，杂草丛生，屋顶还有几处破洞。一婆子站在院门口，满脸不耐烦："四小姐，这就是您的院子了，将就住着吧。"

【核心冲突】
△管家走后，那婆子叉着腰道："四小姐，您可别指望有人伺候您。夫人说了，您是乡下养大的，不懂规矩，先学着点规矩再说。"

【爽点释放】
△萧云灼轻笑一声，从袖中取出一锭金子，在指尖转了转，语气淡淡："规矩我不懂，但银子倒是认得。帮我把这院子修缮好，剩下的——算你的辛苦费。"

【情感拉扯】
（那婆子眼睛都直了，结结巴巴接过金子）她看着萧云灼的眼神瞬间变了，从轻视变成了惊疑不定：这乡下丫头，哪来这么多钱？

【悬念钩子】
→ 萧云灼独自进了破屋，丫鬟绿鹤放下包袱，担忧道："小姐，您为何要装穷？凭您的本事......"萧云灼抬手打断她："不急，时机未到。"
→ 她走到窗前，望向府中某处，喃喃道："这祖祠的位置......倒是有些意思。"

## 第3集

【视觉钩子】
△萧府正门处，一辆华贵的马车停下。一袭月白长袍的青年男子走下马车，眉目温润如玉，腰间悬着象征太子伴读身份的玉佩。萧文愈扫了一眼府中下人，问："四妹妹呢？"

【核心冲突】
△下人支支吾吾，萧文愈脸色渐沉，大步向偏僻角落走去。推开那扇破旧的院门，看见正蹲在地上翻看杂草的萧云灼，眼眶瞬间红了："灼儿......你受苦了。"

【爽点释放】
△萧云灼抬头，露出一个乖巧的笑："大哥放心，我住得惯。"萧文愈一把将她拉起，注意到她手腕上的旧伤，声音发颤："祖母已经骂过母亲了，以后不会再让你受委屈。"

【情感拉扯】
（萧云灼内心os：大哥，你误会了。我只是觉得这里的杂草长得有趣......不过，有个傻子护着的感觉，好像也不坏。）

【悬念钩子】
→ 萧文愈注意到萧云灼身后跟着的丫鬟绿鹤，那丫鬟看他的眼神有些古怪，像是在憋笑。
→ 远处，姜氏正带着一个衣着华贵的少女往这边走来，那少女面若桃花，眼底却藏着算计——正是姜氏的娘家侄女姜妧。

## 第4集

【视觉钩子】
△姜妧挽着姜氏的手臂款款而来，一身藕荷色织金裙，头戴翡翠簪，通身的气派比萧家正经小姐还贵气三分。她看见萧云灼，眼眶微红，哽咽道："表妹，你可算回来了，我日日都在盼着你。"

【核心冲突】
△姜妧上前握住萧云灼的手，眼泪说来就来："表妹在外面受苦了吧？不像我，在姑母身边锦衣玉食......"萧云灼低头看着被她握住的手，眉头微挑。

【爽点释放】
△萧云灼轻轻抽回手，似笑非笑地看着姜妧："表姐这话说得，倒像是我抢了你的福气似的。"她顿了顿，意味深长道："不过表姐这气色......最近可得小心些，莫要往水边去。"

【情感拉扯】
（姜妧脸色微变，强笑道："表妹说笑了。"）萧云灼看着她眼底一闪而过的惊慌，心下了然：果然，有问题。

【悬念钩子】
→ 姜氏冷下脸："云灼，你刚回来就诅咒你表姐，什么规矩？"萧云灼无辜地眨眨眼："母亲，女儿只是好心提醒，算命嘛，宁可信其有。"
→ 萧文愈皱眉挡在萧云灼身前："母亲，灼儿初来乍到，您别总为难她。"

## 第5集

【视觉钩子】
△深夜，萧云灼独自在院中对月饮茶。绿鹤低声禀报："小姐，二少爷的院子今日闹了鬼，下人们都说听见有女子哭声。"萧云灼挑眉："哦？去看看。"

【核心冲突】
△萧云灼悄无声息地出现在二哥萧文远院外，只见院中烛火摇曳，一个披头散发的"女鬼"正贴着窗户往里看。萧文远在屋内瑟瑟发抖，大喊："鬼！有鬼！"

【爽点释放】
△萧云灼抬手一挥，一张符纸飞出，"女鬼"瞬间僵住，摔落在地——竟是个真人假扮的。她蹲下身，扯开"女鬼"脸上的乱发，露出一个尖嘴猴腮的男人。"说说吧，谁让你来的？"

【情感拉扯】
（那男人吓得浑身发抖："是......是姜家表小姐让我来的......她说我装鬼吓二少爷，事成后给我一百两银子！"）萧云灼轻笑：有意思，这姜妧是想嫁祸谁？

【悬念钩子】
→ 萧云灼让绿鹤将人带走，远处，二公子萧文远透过窗户看见了这一幕，眼中闪过精光。
→ 她转身时，目光不经意间扫过府中祖祠的方向，那里的黑气比白日更浓了。

## 第6集

【视觉钩子】
△萧文愈来找萧云灼，却发现她的丫鬟绿鹤正在指挥一群下人搬东西——上好的家具、绫罗绸缎、珠玉摆件流水般送进那破院子。他愣住了："灼儿，这是......"

【核心冲突】
△萧文愈正要追问，姜氏身边的婆子来了，冷着脸道："四小姐，夫人让您去正堂。二少爷说昨晚有人闯进他院子行凶，要您去对质。"萧文愈脸色一沉。

【爽点释放】
△正堂内，姜氏高坐堂上，姜妧在一旁垂泪。萧文远指着萧云灼道："就是你！你派那个丫鬟来害我！"萧云灼慢悠悠地从袖中取出一张纸："二哥说什么？不如我替您算算？"

【情感拉扯】
（萧云灼念出纸上内容："姜家表小姐出资一百两，派刁三装鬼入府。"姜妧脸色煞白）萧文愈冷笑一声，看向姜氏："母亲，您还有什么话说？"

【悬念钩子】
→ 姜氏气得浑身发抖，却碍于证据不敢发作。萧云灼临走时，意味深长地看了姜妧一眼："表姐，水边危险，可得小心。"
→ 当夜，府中传言：四小姐会法术，连鬼都听她的。

## 第7集

【视觉钩子】
△花园中，一个锦衣小男孩带着几个小厮拦住萧云灼的去路。他指着萧云灼大骂道："就是你欺负我表姐！乡下丫头，你算什么东西，敢在萧家撒野？"这是姜氏的小儿子，萧文远一母同胞的弟弟萧文宣，今年才十岁。

【核心冲突】
△萧文宣叉着腰，得意洋洋："我娘说了，你就是扫把星，生你的时候差点害死她，你给我滚出萧家！"绿鹤脸色一变，正要上前。

【爽点释放】
△萧云灼抬手拦住绿鹤，蹲下身，与萧文宣平视，笑得温和："文宣啊，你知道你娘为什么那么喜欢你吗？"她在他耳边轻声说了几句话，萧文宣的脸色瞬间变了。

【情感拉扯】
（萧文宣想起母亲对他说过的话——"你是娘最疼爱的儿子，以后萧家都是你的。"而萧云灼说的是："你娘喜欢的是你哥哥，你只是个备胎。"）他捂着脸跑走了，边跑边哭。

【悬念钩子】
→ 姜氏闻讯赶来，心肝宝贝地哄着萧文宣，扬言要惩罚萧云灼。萧云灼只是笑："母亲，孩子撒谎可不好，我不过是教他做人的道理。"
→ 萧文愈再次出现，将萧云灼护在身后，无奈又心疼地看着她：妹妹又"受委屈"了。

## 第8集

【视觉钩子】
△萧府老太君的院子外，乌云密布，明明是正午，却透着一股阴森之气。萧云灼站在院外，看着那缠绕在屋檐上的黑气，眉头紧锁。绿鹤低声道："小姐，这黑气不对劲。"

【核心冲突】
△正这时，萧文愈匆匆赶来，神色焦急："灼儿，你怎么在这儿？祖母突然晕倒了，大夫们都看过了，找不出病因！"萧云灼没有回答，只是盯着老太君的屋子，眼底闪过一道精光。

【爽点释放】
△萧云灼大步走进屋内，姜氏正在床边哭泣。她绕开众人，径直走到老太君床头，伸手在枕下摸索片刻，取出一柄锈迹斑斑的匕首——刀身上隐隐透着黑气。

【情感拉扯】
（萧云灼冷声道："这刀是谁放在这儿的？"姜氏眼神闪躲："我不知道......"）萧云灼看了她一眼，轻飘飘道："母亲最近睡得可好？这东西放在老太君枕下，可是会折寿的。"

【悬念钩子】
→ 萧云灼当场施法，用符纸化解了匕首上的邪气。片刻后，老太君悠悠转醒，拉着萧云灼的手老泪纵横："好孩子，是你救了我......"
→ 姜氏的脸色白得像纸，眼中闪过一丝狠毒。

【核心冲突】
△绿鹤嘴角抽了抽，努力保持面无表情："大少爷，四小姐在......外面过得很好。"萧文愈一脸不信，眼中满是心疼："你就别为她遮掩了，她肯定是不想让我担心。"

【爽点释放】
△萧文愈拉着萧云灼的手，郑重道："灼儿，以后有什么事尽管告诉大哥。萧家亏欠你的，大哥都会替你讨回来。"萧云灼看着他真挚的眼神，难得沉默了一瞬。

【情感拉扯】
（萧云灼内心os：大哥，你这么认真，我都不知道该怎么告诉你，你妹妹我其实是玄门第一高手......算了，就让你误会着吧，反正也不讨厌。）

【悬念钩子】
→ 萧云灼收到消息：神隐门的人找到了她的踪迹，不日将有人来访。她靠在椅背上，神色淡淡："来得倒是快。"
→ 与此同时，萧府祖祠里，一盏长明灯忽然灭了，守夜的下人吓得魂飞魄散。

## 第10集

【视觉钩子】
△萧府祖祠内，祖宗牌位前的长明灯全部熄灭，香案上供着的贡品全部腐烂发臭，一股腐朽之气弥漫开来。姜氏得知消息后大惊失色："这可如何是好！祖祠出事，萧家要遭大难啊！"

【核心冲突】
△姜氏请来的道士在祖祠里跳了半天大神，非但没用，反而被一阵阴风吹得口吐白沫。府中人心惶惶，流言四起。萧文愈皱眉："母亲，要不请灼儿来看看？"

【爽点释放】
△萧云灼踏进祖祠的那一刻，周身气场骤然一变。她负手而立，目光如炬扫视四周，淡淡道："雕虫小技。"她抬手画符，口中念念有词，刹那间金光大盛，祖祠中的阴气瞬间消散。

【情感拉扯】
（萧文愈看着祖祠中金光环绕的妹妹，目瞪口呆）他喃喃道："灼儿......你什么时候......这么厉害了？"萧云灼回头，对他露出一个乖巧的笑："大哥，我不是说了吗？我在外面过得很好。"

【悬念钩子】
→ 萧云灼在祖祠深处发现了一个隐秘的阵法，正是有人刻意用来吸取萧家气运的。她眸色微沉："看来，有人想让萧家败落。"
→ 当夜，一个黑衣人出现在萧府墙头，望着萧云灼的院子，低声道："门主，神隐门上下恭候您的归来。"

# 《玄门神算卦太灵，火爆全京城！》

## 第11集

### 【视觉钩子】
△深夜，萧云灼独坐窗前，月光洒落在她苍白的脸上。桌上铜钱轻轻震动，卦象浮现——一个血红凶字。

### 【核心冲突】
△"二少爷深夜出府，去了东城赌坊。"丫鬟低声禀报。
△萧云灼指尖轻点卦象："赌？不像……他被人牵着鼻子走。"

### 【爽点释放】
△萧云灼带人直闯赌坊，正好撞见二哥萧文轩被人按在地上，欠下三千两赌债。
△她不慌不忙从袖中取出一沓银票："三千两？太少了，我替他赌——"
△三把骰子，把把通杀。赌坊老板脸色铁青，反而欠她五千两。
△"钱送到萧府，慢走不送。"她扬长而去，留下萧文轩目瞪口呆。

### 【情感拉扯】
→ 内心os：这个蠢货二哥虽然可恨，但若被外人大赌坊套进去，整个萧府都会遭殃。罢了，就当替祖母省心。
→ 大哥萧文愈追上来，满脸复杂："阿灼，你怎么知道……"
→ "猜的。"她微微一笑。

### 【悬念钩子】
→ 钩子1：赌坊幕后之人是谁？为何针对萧文轩？
→ 钩子2：萧云灼离开时，一个蒙面人正盯着她的背影——那人腰间挂着皇室专用的玉佩。

## 第12集

### 【视觉钩子】
△清晨，萧府门口停着一顶华丽轿子。姜氏的娘家嫂子——姜夫人趾高气扬地下了轿，身后跟着姜妧。

### 【核心冲突】
△"外甥女回来这么大事，怎么不通知我们姜家？"姜夫人皮笑肉不笑，"还是说，有人故意瞒着？"
△姜氏脸色微变，看向萧云灼的眼神更加厌恶。

### 【爽点释放】
△姜妧上前亲热地挽住萧云灼的手："表妹这身衣裳……是在哪儿买的？款式真别致呢。"
△话音未落，萧云灼腕上的翡翠镯子滑落，"不小心"砸在姜妧脚边——这是姜氏偷给姜妧的萧家传家宝。
△"哎呀，表妹怎么摔我的镯子！"姜妧尖叫。
△萧云灼挑眉："你的？萧家族谱上，这镯子可是记在我娘亲名下。"
△姜氏脸色煞白。

### 【情感拉扯】
→ 内心os：以为我是软柿子？上辈子你们姜家可没少占萧家便宜，这一世，一文钱都别想拿走。
→ 大哥萧文愈皱眉："什么镯子？"
→ 萧云灼淡淡道："没什么，一只不请自来的……白眼狼。"

### 【悬念钩子】
→ 钩子1：姜氏偷拿萧家传家宝给娘家侄女，被揭穿后如何收场？
→ 钩子2：姜夫人此次来访，真正的目的是什么？

## 第13集

### 【视觉钩子】
