# 等一场盛夏

## 第1集：消失的星星

【视觉钩子】
△深夜，某颁奖典礼后台，璀璨灯光骤然熄灭，大屏上滚动着“佑白”二字。一个戴着银色口罩的身影独自穿过人群，身后是无数粉丝撕心裂肺的呐喊。他头也不回，消失在黑暗中。三年后，那个让整个乐坛颤抖的名字，再无音讯。

【核心冲突】
△某顶流经纪公司会议室内，总裁焦躁踱步："找到他了吗？"
△助理低头："祁佑……档案是空的。"

【爽点释放】
△某音乐学院荣誉墙上，“祁佑”两个字金光熠熠——全国高考理科状元，竞赛金牌无数。
△旁边贴着十年前的旧报纸：《天才少年横扫音乐颁奖礼！》。

【情感拉扯】
（旁白）：所有人都以为那个夏天之后，佑白彻底消失了。

【悬念钩子】
→ 没有人知道，那个让乐坛封神的少年，现在是一所普通高中里最嚣张的“问题学生”。
→ 没有人知道，他改了名字，顶着一头银发，坐在最后一排，眼神冷漠得像另一个人。

## 第2集：初来乍到

【视觉钩子】
△九月的阳光刺眼，一辆破旧的面包车停在校门口。穿着白裙的少女拖着行李箱，额发被汗水浸湿，狼狈地蹲在地上喘气。周围是来来往往的名车和衣着光鲜的学生，她像误入天鹅湖的丑小鸭。

【核心冲突】
△夏时（小声）："请问……女生宿舍怎么走？"
△几个女生掩嘴笑："这年头还有人走错学校？"
△一辆黑色越野车轰鸣而来，车窗摇下一半——银色碎发，冷白的手指点着太阳穴，语气懒散。

【爽点释放】
△祁佑（瞥她一眼）："麻烦。"
△下一秒，他面无表情地从车上扔下一瓶矿泉水，砸在夏时脚边。
△"接着。"

【情感拉扯】
（夏时内心OS）：这人好凶……但好像也不是坏人？

【悬念钩子】
→ 夏时捡起水瓶，指尖触到冰凉的瓶身，愣在原地。
→ 那个眼神，好像在哪里见过……

## 第3集：那个名字

【视觉钩子】

△祁佑突然抬头，正好对上夏时的视线。

【爽点释放】

△祁佑（冷漠脸）："不认识。"

△两人异口同声。

【情感拉扯】

【悬念钩子】
→ 夏时收拾书包时，一本书掉在地上，封面上赫然写着——佑白·《盛夏》。

## 第4集：你的偶像

【视觉钩子】

【核心冲突】
△女生A（翻白眼）："佑白？谁啊？三线小糊咖？"
△夏时（攥紧手机）："他不是……他是很厉害的人……"
△教室后排传来椅子刺耳的摩擦声。

【爽点释放】
△祁佑不知道什么时候走了过来，拿起夏时的手机看了看，嘴角勾起一抹意味不明的弧度。
祁佑："眼光不错。"
△女生A："祁佑学长你也知道这个人？"
祁佑（挑眉）："不知道。随便说说。"

【情感拉扯】
（夏时内心OS）：他……为什么会看我的手机壳？
（祁佑内心OS）：这个笨蛋，还留着那张专辑。

【悬念钩子】

→ 她抬头看向最后一排，祁佑已经不知所踪。

## 第5集：食堂风波

祁佑："挡着点，丢人。"

【情感拉扯】

## 第6集：夜半琴声

【视觉钩子】
△深夜，夏时因为睡不着，独自走在校园里。音乐楼的某个教室亮着灯，隐约传来钢琴声。

【核心冲突】
△夏时悄悄靠近，从门缝看进去——
△祁佑坐在钢琴前，银发垂落，修长的手指在黑白键上飞舞。那旋律熟悉得让她心颤。

【爽点释放】
△《盛夏》的前奏响起——这是佑白最经典的曲目，也是夏时听了无数遍的歌。
△夏时（喃喃）："佑白……"
△琴声戛然而止。

【情感拉扯】
△祁佑转过头，目光幽深地看着门缝中的夏时。
祁佑（冷冷地）："你听到了多少？"
△夏时慌乱后退，撞翻了门口的椅子。

【悬念钩子】
→ 祁佑一步步逼近，把夏时逼到墙角。
→ 祁佑（声音低沉）："记住，今晚什么都没听到。"

## 第8集：粉丝见面会

△夏时坐在台下，攥紧了手里的荧光棒，眼眶泛红。

【核心冲突】
△夏时疯了一样冲向音乐教室。
△祁佑正站在窗边，逆光中他的轮廓与记忆中那个舞台上的少年重叠。

【爽点释放】

【情感拉扯】

祁佑（声音沙哑）："你不知道你在说什么。"
夏时（倔强地看着他）："我知道。我一直都知道。"
△祁佑的眼神终于出现裂痕。

【悬念钩子】

## 第10集：等一场盛夏

【视觉钩子】

【核心冲突】
△主持人："有请学生代表祁佑发言。"
△祁佑走上台，却从口袋里掏出一把吉他。
祁佑："三年前，我是一个懦夫。今天，我想唱一首歌。"

【爽点释放】
△吉他声响起，是《盛夏》的前奏。
△全场哗然。
△祁佑的目光穿过人群，看向站在角落的夏时。
祁佑（轻声唱）："那年夏天，我以为世界抛弃了我……"

【情感拉扯】
△夏时泪流满面，却在人群中拼命点头。
△祁佑唱着唱着，声音哽咽了。
祁佑（看向夏时）："直到有个人告诉我——星星只是暂时闭上了眼睛。"

【悬念钩子】
→ 歌曲结束，全场寂静。
→ 祁佑放下吉他，一步一步走向夏时。
→ 祁佑（伸出手）："夏时，我等了你三年。"

## 尾声：盛夏终章

【视觉钩子】

【核心冲突】
△夏时挺着微微隆起的肚子，被祁佑牢牢护在怀里。

祁佑（傲娇）："人太多。"

【爽点释放】
△记者蜂拥而上："祁佑，请问您复出的契机是什么？"
△祁佑（看向夏时）："因为我答应一个人，要让全世界重新听到我的声音。"
△夏时脸红，低头轻抚小腹。

【情感拉扯】
△祁佑俯身，在夏时额头落下一吻。
祁佑（低声）："谢谢你，等到了我的盛夏。"

【悬念钩子】
→ 舞台上灯光骤然亮起。
→ 全场粉丝疯狂尖叫。
→ 祁佑握着夏时的手，一步一步走向那个属于他的舞台。
→ （旁白）：有些星星，注定要等到属于它的盛夏，才会再次闪耀。

## 第十一集

【视觉钩子】

【核心冲突】
△“喂。”祁佑不知何时走到她身边，压低声音，“你占我位置了。”
△夏时抬头，一脸茫然：“这里写的是'无人'啊？”
△祁佑面无表情地把一张纸条拍在她面前——“祁佑专属”。他故意用张扬的字迹写着。

【爽点释放】
△夏时正要道歉，祁佑却已经把书包往旁边座位一放：“既然你坐了，就别换地方了。”
△她困惑地看着他，却见他耳尖微红：“……这里光线好，你别多想。”

【情感拉扯】
（夏时内心OS：他……是特意来坐我旁边的？不对，他明明说占位置了，所以是在赶我走？还是在留我？到底什么意思啊……）
△祁佑装作若无其事地打开课本，余光却始终黏在她身上。他发现她做笔记时喜欢咬笔帽，便皱着眉把自己的笔递过去：“用这支，别咬那个脏的。”

【悬念钩子】
→ 夏时的手机突然震动，是小姨发来的消息：“时时，祁家那边的人好像在打听你，你小心点。”
→ 祁佑的手机也同时亮起，他瞥了一眼屏幕，神色瞬间冷了下来。

## 第十三集

【情感拉扯】

【悬念钩子】

→ 夏时没注意到他的异样，还在兴奋地说着：“他五年前突然消失，我真的好难过……”

【核心冲突】
△夏时因为落下的笔记本折返回来，透过门缝看到了这一幕。她捂住嘴，不敢相信自己看到的——那个弹琴的身影，和她记忆里“佑白”的演出视频重叠。

【爽点释放】
△祁佑弹完最后一个音符，却没有转身。他像是自言自语：“有些歌，注定只能弹给自己听。”
△夏时正要推门进去质问他，祁佑的手机突然响起。他接起电话，语气冰冷：“我说过了，不复出不接受采访。那些人听不懂人话？”
△电话那头传来经纪人的哀求声：“佑白，公司已经妥协了，你到底要怎样才能回来？粉丝们都在等你啊……”

【情感拉扯】
（夏时僵在门外，脑海里一片空白。祁佑……就是佑白？那个她追了好多年的偶像？可是他明明……为什么会是同一个人？）
△她踉跄后退，却不小心撞到了门框，发出声响。祁佑猛地回头，两人的视线隔着门缝对上。

【悬念钩子】
→ 祁佑的脸色瞬间变了，他挂断电话，快步走向门口。
→ 夏时转身想跑，却被追出来的祁佑一把抓住手腕。

## 第十五集

【视觉钩子】
△月光下的天台，祁佑将夏时拉到角落里。他松开手，靠在栏杆上，神情是从未有过的疲惫和警惕。

【核心冲突】
△“你听到了多少？”祁佑的声音很冷，和平时的傲娇完全不同。
△夏时攥紧衣角：“我……我只是看到你在弹琴，那首曲子，是佑白的成名曲。你……你是佑白吗？”
△祁佑沉默了片刻，忽然笑了，笑容里带着自嘲：“是啊，我就是那个抛下粉丝逃跑的懦夫。怎么，失望了？”

【爽点释放】
△夏时愣了几秒，然后做了一件祁佑完全没想到的事——她哭了，眼泪大颗大颗地往下掉。
△祁佑慌了：“喂，你哭什么？是我在骂自己又不是在骂你……”他手忙脚乱地想帮她擦眼泪，又缩回去，反复好几次。
△夏时抽噎着说：“我找了你五年……你知不知道我多担心你……你为什么什么都不说就消失了……”

【情感拉扯】
（祁佑内心动摇：她……她在为我哭？五年了，从来没有人这样问过我为什么要离开。所有人只在乎我能不能回来，没有人问过我为什么要走……）
△他终于伸出手，轻轻把她的脑袋按进自己怀里：“别哭了，吵死了。”声音却带着一丝颤抖。
△夏时埋在他胸口，感受到他心跳得很快，和他故作冷漠的语气完全不同。

【悬念钩子】
→ 祁佑的声音在她头顶响起，带着从未有过的认真：“夏时，我的过去很复杂，比你想象的还要乱。你确定……要继续靠近我这样的人吗？”
→ 远处突然传来脚步声，似乎有人在向天台靠近。祁佑下意识地把夏时护在身后，神色警惕：“躲好。”
