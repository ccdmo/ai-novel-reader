# 前女友死了十年，还给我发消息？

## 第1集

### 【视觉钩子】

△精神病房内，主角陆离安静地坐着，瞳孔中倒映出主治医生的身影——那医生的嘴正在撕咬着什么，血肉模糊。走廊里护士的影子扭曲变形，墙壁渗出黑色粘液。

### 【核心冲突】

△医生微笑着递来药片："陆先生，该吃药了。"
△陆离看着药片，声音平静："又是控制妄想的药？可我觉得，真正有病的是你们。"
△他低头看向自己的手——指尖隐约有墨青色微光一闪而过。

### 【爽点释放】

△陆离突然抓住医生的手腕，将他的手翻过来——手背上一道细长的黑色裂纹正在蠕动。
△他轻声道："医生，你的'妄想症'也很严重啊。"
△医生脸色骤变，瞳孔收缩成竖线。

### 【情感拉扯】

（陆离内心OS：这半年了，这些'幻觉'越来越清晰。到底是我疯了，还是这个世界疯了？）
（叶谨言……你怎么突然发消息来？）

### 【悬念钩子】

→ 手机震动，一条消息弹出
→ 备注名：叶谨言
→ "你把我撞死了，到底还打不打算赔偿？"

## 第2集

### 【视觉钩子】

△精神病院大门缓缓打开，陆离独自走出来。阳光刺眼，他眯起眼睛。
△身后的医院建筑在光线下显得格外正常，但他看见——楼顶有什么东西在蠕动。

### 【核心冲突】

△陆离家门口，妹妹陆小听冲出来抱住他，眼眶通红。
△"哥！你终于回来了！我好担心你！"
△陆离僵住："……小听，我走了半年，你不说我上个月才辞职吗？"

### 【爽点释放】

△陆小听愣住，摇头："哥，你在说什么？我们上周还一起吃饭啊。"
△陆离翻出手机通话记录——确实只有几天前的通话。
△他看向妹妹身后——她的影子，正在朝他诡异地微笑。

### 【情感拉扯】

△陆离下意识把妹妹拉到身后，喉咙发紧："小听，你最近……有没有觉得哪里不对劲？"
△陆小听困惑地看着他："哥，你怎么了？你撞的那个人不是已经——"
△"什么？"

### 【悬念钩子】

→ "我"撞了人？什么时候？我只记得去看病……
→ 妹妹口中的"那个人"是谁？

## 第3集

### 【视觉钩子】

△夜晚，陆离坐在自己房间里，周围的阴影似乎在呼吸。
△桌上放着那盒没吃的药片，他犹豫了一下，把药片丢进抽屉。

### 【核心冲突】

△陆离打开手机，盯着"叶谨言"的聊天框，手指颤抖。
△他终于打出一行字："你是谁？"
△对面秒回："你前女友。"

### 【爽点释放】

△陆离追问："你死了十年，怎么发消息？"
△叶谨言："因为是你杀了我。"
△他猛地站起来——等等，我杀了谁？十年前我根本没有女朋友！

### 【情感拉扯】

△陆离呼吸急促，回想十年前的记忆——一片空白。
（不可能……我连她长什么样都不记得……可为什么看到这个备注名，心脏会这么疼？）

### 【悬念钩子】

→ 陆离决定查十年前的新闻
→ 屏幕上跳出一条旧报道：《女大学生深夜遭撞，肇事司机逃逸》

## 第4集

### 【视觉钩子】

△陆离盯着新闻报道，照片模糊，但那个名字清晰可见——叶谨言，21岁，美术系学生。
△他的手指悬在屏幕上，无法动弹。

### 【核心冲突】

△他继续往下翻，找到自己当年的社交账号——没有任何和"叶谨言"相关的内容。
△但他打开相册，发现一张被加密的照片，密码提示是：她的生日。
△他不知道她的生日。

### 【爽点释放】

△陆离试着输入一个日期——手机解锁了。
△照片加载出来：阳光下的女孩，笑着朝镜头比耶。
△他完全不认识这张脸，但眼泪突然涌了出来。

### 【情感拉扯】

△陆离捂住脸，肩膀颤抖。
（为什么……我不记得你，却这么难过？）
△手机再次震动，叶谨言发来一条语音。

### 【悬念钩子】

→ 语音内容是什么？
→ 陆离为什么会忘记她？

△"你说会永远记得我。"

### 【爽点释放】

### 【情感拉扯】

### 【核心冲突】

△陆离下载安装，屏幕弹出提示："检测到心灯宿主，欢迎进入诡境。"

△下一秒，他被拉入一个昏暗的宴会厅。

### 【爽点释放】

△周围穿着华丽的"宾客"齐刷刷转头看他，眼睛空洞，瞳孔是竖线。
△一个穿着龙袍的"皇子"站起来，微笑："欢迎来到我的游戏，陆离先生。"
△"找到你身边的诡异，或者——成为诡异的养料。"

### 【情感拉扯】

△陆离环顾四周，看见角落里站着一个熟悉的身影——
△是叶谨言。但她没有笑容，眼角有泪痕。
△"陆离，你终于来了。"

### 【悬念钩子】

→ 叶谨言怎么会在这里？她是什么身份？
→ "二皇子"的游戏规则是什么？

# 《前女友死了十年，还给我发消息？》

## 第11集

【视觉钩子】
△深夜，陆离独坐书房，手机屏幕的冷光照亮他苍白的面孔。屏幕上，叶谨言的消息还停留在那句“你把我撞死了”。
△心口的墨青色印记隐隐发烫——那是心灯的种子。
△他抬起头，瞳孔深处倒映出窗外一道扭曲的黑影。

【核心冲突】
△陆离决定回复叶谨言的消息：“你是谁？”
△回复的瞬间，窗外黑影猛然贴近玻璃——一张扭曲的人脸贴在外面，正对着他笑。
△“你不该问的。”手机屏幕突然跳出叶谨言的新消息。

【爽点释放】
△黑影破窗而入的瞬间，陆离心口墨青色印记爆发火光——他本能地抬起手，一道墨青色火焰激射而出。
△火焰精准命中黑影，将那扭曲的存在直接吞噬殆尽。
△碎片散落一地，陆离低头，发现地上是一张扭曲的照片残片——照片上的人，竟是他自己。

【情感拉扯】
（我看着手心残留的墨青色火焰，心跳快得像要从胸腔里蹦出来。）
（这不是幻觉。这他妈不是幻觉！）
（那十年……那辆车……叶谨言……她到底是谁？）

【悬念钩子】
→ 照片残片上的另一个“我”是谁？为何长得一模一样？
→ 叶谨言发来最后一条消息：“三天后，老地方见。别迟到。”

## 第12集

【视觉钩子】
△陆离翻遍家中所有旧物，找到一个尘封的铁盒。盒盖上贴着一张褪色的照片——两个年轻人在海边笑着。
△照片里的女生，正是叶谨言。
△但照片背面写着一行字：“2014年7月14日，永远的谎言。”

【核心冲突】
△他调出手机里的旧照片对比——十年前他和叶谨言的合照，和这张照片一模一样。
△但他的记忆里，叶谨言是2015年才出车祸去世的。
△2014年？他们在海边？发生了什么？

【爽点释放】
△陆离用新燃起的心灯能力扫描那张旧照片——照片表面浮现出肉眼看不见的字迹：
△“陆离，记住，是我让你忘记的。”
△字迹如活物般扭曲重组，最终显示出一串地址。

【情感拉扯】
（我的手指在发抖。她说“是我让你忘记的”……）
（她到底为我做了什么？为什么我什么都不记得？）
（不对……如果她让我忘记，为什么现在又让我想起来？）

【悬念钩子】
→ 叶谨言十年前到底发生了什么？她为什么要让陆离“忘记”？
→ 地址指向的地方，隐藏着什么真相？

## 第13集

【视觉钩子】
△陆离按照地址来到城郊一座废弃的精神疗养院——正是他曾被“诊断”为妄想症的地方。
△疗养院大门紧闭，但围墙上有新鲜的脚印。
△他翻墙而入，心灯自动在瞳孔中燃烧，照亮了黑暗中隐藏的无数扭曲身影。

【核心冲突】
△疗养院内，数十个“诡异”正在游荡——它们都穿着病号服，面容模糊。
△陆离发现墙上贴满了病人档案，每一份档案的照片……都是他认识的人。
△包括他自己。
△一份档案掉落脚下：“患者编号001，陆离。诊断：认知过载，建议重置记忆。”

【爽点释放】
△陆离冷笑一声，抬手释放心灯火焰。
△墨青色火海席卷整个走廊，所有诡异发出凄厉尖啸，被尽数吞噬。
△火焰燃烧间，他的脑海中涌入大量记忆碎片——2014年7月14日，他亲眼看到叶谨言被一辆黑色轿车撞飞。
△而那辆车的车牌，正是他自己的车牌。

【情感拉扯】
（不……不可能……）
（我杀了她？十年前我杀了她？）
（那这十年……我为什么会忘记？是谁让我忘记的？）

【悬念钩子】
→ 如果陆离十年前真的杀了叶谨言，那现在发消息的“叶谨言”究竟是谁？

## 第14集

【视觉钩子】
△疗养院地下室，一扇锈迹斑斑的铁门后，是一间布满仪器的房间。
△房间中央的病床上，躺着一个女人——面容与叶谨言一模一样，但明显已经死亡多年。
△她的身体上连接着数十根管子，管的另一端……连接着整个城市的地图模型。

【核心冲突】
△陆离僵在原地。床头的病历写着：“叶谨言，2014年7月14日入院，诊断：意识分裂。”
△“她在用自己喂养这座城市里所有的诡异。”身后传来一个男人的声音。
△陆离猛然回头——站在门口的人，竟然是精神病院的主治医生。那个他曾以为是“诡异”的男人。

【爽点释放】
△医生摘下眼镜，瞳孔中燃烧着同样墨青色的火焰。
△“心灯者，我们都以为你死了。十年前那场'重置'，本该把你一起抹杀。”
△陆离冷声问：“谁重置的？”
△医生看向病床上的叶谨言：“她。”

【情感拉扯】
（她用自己喂养诡异？她……为了救我，用自己换了这座城市十年的平安？）
（那我呢？我什么都不记得，我活了十年，她却死了十年……）
（她他妈凭什么替我做这个决定！）

【悬念钩子】
→ 叶谨言十年前到底做了什么？为什么要“重置”陆离的记忆？
→ 医生接下来的话更令人震惊：“你前女友还活着。她现在……在诡境等你。”

## 第15集

【视觉钩子】
△医生递过来一面古旧的铜镜，镜面漆黑如墨。
△“这是通往诡境的入口。你要见她，就进去。但我警告你——诡境里养着这座城市所有的'养诡人'，包括二皇子的人。”
△陆离接过铜镜，镜面突然浮现出叶谨言的面容——她在镜中对他微笑。

【核心冲突】
△陆离踏入镜面的瞬间，天旋地转。
△他站在一片荒芜的城市废墟中——这里的天空是倒挂的，建筑是扭曲的。
△远处，一个熟悉的身影站在废墟中央。
△是叶谨言。活着的叶谨言。
△她穿着十年前的白色连衣裙，正对着他笑：“你来晚了，陆离。”

【爽点释放】
△陆离冲上前想抓住她，却在触碰的瞬间穿透而过——那只是残影。
△真正的叶谨言站在他身后，声音冰冷：“傻子，我早就死了。这是我的意识残片，专门等你来收的。”
△陆离猛然回头，看到叶谨言手中握着一颗跳动的墨青色心脏——心灯的最终形态。
△“拿去吧，杀了我，你就能点燃完整的心灯。然后……替我杀了二皇子。”

【情感拉扯】
（我看着她的笑容，眼泪不受控制地流下来。）
（十年了，她一直在等我。等我来杀她。）
（她笑着说“拿去吧”，可她的手在发抖，你知道吗？）

【悬念钩子】
→ 叶谨言为何要陆离杀了自己？二皇子又是谁？
→ 如果陆离真的杀了叶谨言……他能承受这个代价吗？
→ 叶谨言最后说了一句：“记得，那天撞我的车，不是你的。你只是……替罪羊。”

# 《前女友死了十年，还给我发消息？》

## 第16集

【视觉钩子】

△深夜，客厅一片漆黑。陆离的手机屏幕亮起，映照着他苍白的脸。屏幕上，叶谨言的对话框显示着新消息："你以为躲得掉吗？"

△镜头缓缓拉近，陆离的眼眶泛红，喉结剧烈滚动。他颤抖的手指悬在屏幕上方，迟迟不敢点开。

【核心冲突】

△陆离（低声呢喃）："十年……你已经死了十年……"

△手机震动，新消息弹出："躲不掉的，陆离。你杀了我，就必须偿还。"

△陆离猛地站起身，椅子被撞翻在地。他的呼吸急促而紊乱，额头渗出冷汗。

【爽点释放】

△陆离突然冷笑一声，眼神从慌乱转为锐利："你说你死了十年？好，我信。那我就查清楚——你到底是怎么死的。"

△他打开电脑，指尖飞快敲击键盘："十年前……叶谨言……车祸……"

△搜索结果跳出：十年前青城市一中门口发生一起车祸，女高中生当场死亡，肇事司机至今未被找到。

△陆离瞳孔骤缩——那所中学，正是他当年就读的学校。

【情感拉扯】

△陆离（内心os）："我杀了她？我亲手杀了她？可我为什么……一点都不记得？"

△他的手按在胸口，那里隐隐作痛。十年前，他到底做过什么？

△窗外，夜风吹动树叶发出沙沙声响。陆离猛地回头——窗户上，映出一张模糊的人脸。

【悬念钩子】

→ 窗户上的脸越来越清晰，是叶谨言的脸，正在对他微笑。

→ 陆离的手腕突然发烫——心灯在体内躁动，像是要破体而出。

## 第17集

【视觉钩子】

△窗户上的叶谨言脸庞渐渐清晰，她的嘴唇开合，发出无声的质问。

△陆离的眼瞳中燃起墨青色火焰，心灯自动点燃。他看见了——那张脸不是投影，而是贴在窗户外面的实体。

△特写：一张苍白的、眼角挂着血泪的女人面孔。

【核心冲突】

△陆离（沉声）："你是诡异……还是她？"

△叶谨言的脸缓缓后退，消失在黑暗中。下一秒，她的声音从四面八方传来——

△叶谨言（画外音）："我死了十年，你也逃了十年。陆离，该还债了。"

△陆离环顾四周，心灯的视野中，整个房间弥漫着淡淡的诡气。

【爽点释放】

△陆离突然笑了，笑声低沉："有意思……你说我撞死了你？好，那我就顺着这条线查下去。"

△他拿出手机，打开叶谨言的对话框："你说你是我前女友，那我们来对一下——你记得什么？"

△消息发出，对面沉默了三秒。

△叶谨言回复："你最喜欢的歌，是《夜曲》。你说过，你最爱的人已经死了。"

△陆离的手猛地一颤——这是他从未告诉过任何人的秘密。

【情感拉扯】

△陆离（喃喃自语）："不可能……这首歌我只唱给一个人听过……"

△记忆碎片闪过：高中毕业典礼的夜晚，他抱着吉他，在天台上为一个人唱了这首歌。

△那个人……是谁？

△陆离的太阳穴剧烈跳动，记忆像是被什么东西封印住了一样。

【悬念钩子】

→ 第二天清晨，陆离收到一封匿名邮件，附件是一张照片：十年前，他站在一辆车前，车的引擎盖上沾满鲜血。

→ 照片下方有一行字："这就是证据。陆离，你逃不掉的。"

## 第18集

【视觉钩子】

△陆离站在十年前事故发生地——青城市一中门口。这里已经变成了一片废墟，但那条斑马线依稀可辨。

△他的目光落在马路对面，那里有一家已经倒闭的奶茶店。招牌上依稀可见两个字："心声"。

△心灯在体内微微发热，周围的诡气浓度骤然上升。

【核心冲突】

△陆离走进废墟，来到奶茶店门口。他推开门，灰尘扑面而来。

△墙上贴着一张褪色的海报：某届高中歌唱比赛的宣传页。一等奖获得者的照片已经模糊，但名字清晰可见——"陆离"。

△陆离皱眉："我参加过这个比赛？"

△角落里，心灯突然剧烈跳动。地上有一道暗门，被灰尘和杂物覆盖。

【爽点释放】

△陆离用力推开暗门，下面是一条幽深的地下通道。心灯的墨青色火焰照亮了前方。

△通道尽头是一间密室，墙上贴满了照片和报纸剪报。全是关于十年前那场车祸的报道。

△最显眼的位置，放着一个相框。相框里的照片——是高中时期的陆离和一个女孩的合照。

△照片背面写着："陆离&叶谨言，2009年夏。"

△陆离的瞳孔骤缩："叶谨言……我高中时的女朋友？"

【情感拉扯】

△记忆洪流般涌来：他和叶谨言是高中恋人，毕业那天，他为她唱了一首歌，承诺永远在一起。

△然后……然后发生了什么？

△陆离头痛欲裂，记忆在这里断裂。他只记得毕业典礼后，他喝醉了，开了一辆车……

△陆离（颤抖）："我酒后驾车……撞死了她？"

【悬念钩子】

→ 就在这时，密室的灯突然亮起。一个苍老的声音响起："你终于来了，陆离。我等你等了十年。"

→ 阴影中，一个穿着病号服的老太太缓缓走出——那是精神病院里给陆离开药的医生。

## 第19集

【视觉钩子】

△密室灯光昏暗，老太太的身影被拉得很长。她的眼睛在黑暗中泛着诡异的竖瞳光芒。

△陆离下意识后退一步，心灯在体内疯狂燃烧。

△老太太（微笑）："怎么，不认识我了？王医生，你的'主治医师'。"

【核心冲突】

△陆离（冷笑）："你们到底是谁？精神病院、养诡、还有十年前的车祸……全是一盘棋？"

△老太太缓缓鼓掌："聪明。不愧是能点燃心灯的人。"

△她走到墙边，指着那些照片："十年前，你确实杀了叶谨言。但不是'她'要找你报仇——是有人要借她的手，杀了你。"

△陆离瞳孔骤缩："什么意思？"

【爽点释放】

△老太太转身，脸上的皮肤开始龟裂，露出下面墨青色的鳞片。

△老太太："养诡者的游戏，听说过吗？每个'天选者'身边，都会有一只'寄生诡异'。他们让诡异杀死天选者，然后收割诡异的力量。"

△她伸出手，指向陆离的胸口："你的心灯……就是他们想要的猎物。"

△陆离（大笑）："想要？那就来拿！"

△墨青色火焰从陆离体内喷涌而出，将整个密室照得通明。老太太的诡异之身被火焰灼烧，发出凄厉的尖叫。

△陆离（冷声）："养诡者？不好意思，我更喜欢——猎诡者。"

【情感拉扯】

△火焰中，陆离的眼前闪过叶谨言的笑容。

△陆离（内心os）："叶谨言……你到底是被谁害死的？我杀了你，他们却利用你的死来杀我……"

△愤怒与悲伤交织，他的火焰燃烧得更加猛烈。

【悬念钩子】

→ 老太太被烧成灰烬的瞬间，她的声音在密室回荡："你杀了我，没用的……真正的养诡者，在'彼岸'。去找叶谨言的墓，她在等你。"

→ 陆离的手机突然震动，是叶谨言的消息："来见我。就在你脚下。"

## 第20集

【视觉钩子】

△密室地面突然裂开，露出一条向下延伸的石阶。幽蓝色的光芒从深处透出。

△陆离深吸一口气，沿着石阶走下去。心灯的墨青色火焰与幽蓝光芒交织，照亮了一条通往地下的通道。

△墙壁上刻满了古老的符文，每走一步，符文就亮起一盏。

【核心冲突】

△石阶尽头是一座地下墓室。正中央有一口石棺，棺盖半开，里面躺着一个人——叶谨言。

△她穿着十年前的校服，面容安详，仿佛只是睡着了。

△陆离（颤抖着走向石棺）："叶谨言……真的是你……"

△就在这时，叶谨言睁开了眼睛。她的瞳孔是竖线，嘴角勾起诡异的弧度。

【爽点释放】

△叶谨言（坐起身，微笑）："陆离，你终于来陪我了。"

△陆离没有后退，反而迎上前去。他伸出手，轻轻触碰她的脸颊。

△陆离（温柔）："你等了十年……我也等了十年。"

△叶谨言的表情瞬间僵住。她感知到——陆离的体内没有一丝恐惧，只有……释然。

△陆离："我杀了你，我来偿命。但在我死之前，我要知道真相——你到底是怎么死的？"

△叶谨言的眼泪突然流下，那是真实的、温热的泪水。

【情感拉扯】

△叶谨言（哽咽）："你不记得了吗？那场车祸……你确实撞死了一个人。但不是我——是我妹妹，叶无言。"

△陆离瞳孔骤缩："什么？"

△叶谨言："十年前，是我妹妹借用了我的身体，想救你……她用自己的命，换了你的命。"

△她的身影开始变得透明："而我……留在人间，是为了完成她的遗愿——让你活下去。"

【悬念钩子】

→ 叶谨言的身形消散，只留下一枚刻着"心"字的玉佩落在石棺上。

→ 陆离拾起玉佩的瞬间，心灯剧烈震动——百日筑基的倒计时突然加速，还剩67天。

→ 手机震动，新消息弹出。发送者是一个陌生号码："恭喜你找到第一块碎片。现在，游戏正式开始。下一个线索——去查'彼岸'集团。"
