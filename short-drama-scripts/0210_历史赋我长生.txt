# 历史赋我长生，我终苟成万朝元老

## 第1集

**【视觉钩子】**
△金銮殿内龙涎香袅袅，年迈的景武帝端坐龙椅，眉眼间尽是帝王的阴鸷。殿中跪着一位身着绯色官袍的青年——他正是起居舍人顾长安。青年低眉顺眼，执笔的手却稳如磐石。

**【核心冲突】**
△景武帝突然暴怒，将奏折砸向跪地的太子："逆子！你敢结交外藩？"
△帝王目光扫向角落里的史官，声音如冰："顾舍人，朕与太子的话，你可都记下了？"
△顾长安叩首，声线平稳："回陛下，臣只记明君贤臣之语。"

**【爽点释放】**
△景武帝眉头微蹙，却冷哼一声："倒是识趣。"摆手示意他退下。
△顾长安从容起身，执笔退至殿侧，嘴角勾起一抹极淡的弧度——这已是他辅佐的第三位皇帝。

**【情感拉扯】**
（内心os：又过了一关。帝王心术，不过如此。再烈的火，也烧不到一个'什么都看不见'的史官身上。）

**【悬念钩子】**
→ 顾长安袖中藏着一卷前朝秘史，那是他三百年来积攒的保命底牌。
→ 太子临退朝时，那意味深长的一瞥，预示着另一场风暴即将降临。

## 第2集

**【视觉钩子】**
△夜幕下的史馆烛火摇曳，满墙都是历朝起居注。顾长安独坐案前，从怀中取出一个瓷瓶，倒出些许灰白粉末，对着铜镜细细涂抹于发鬓与面颊。

**【核心冲突】**
△镜中人原本年轻的面容逐渐变得苍老憔悴，发丝染上霜白。
△门外来人通报："大人，丞相府派人送来拜帖，请您明日过府一叙。"
△顾长安眼皮都没抬："回了。就说我风烛残年，不胜奔波。"

**【爽点释放】**
△来人面露不屑离去后，顾长安轻嗤一声："丞相？这位已是本朝第五位丞相了……前四位，三个砍头，一个流放。"
△他翻出标注"大景·卷三十七"的档案，上面详细记录着每一位权臣的崛起与覆灭。

**【情感拉扯】**
（内心os：他们总想拉史官下水，却不知我已在这池子里游了三百多年。看他们起高楼，看他们楼塌了……何必呢。）

**【情感拉扯】**
（内心os：这灰粉是特制的，涂上后看着行将就木，实则透气舒适。活得太久，总得学会骗人。）

**【悬念钩子】**
→ 丞相急召史官，究竟是想抹去什么起居注中的记录？
→ 顾长安那泛黄的档案袋里，还藏着多少前朝秘辛？

## 第3集

**【视觉钩子】**
△东宫寝殿内，太子萧珩负手而立，烛光勾勒出他清俊却阴沉的面容。案上摊着一封密信，落款处是一枚暗红印章。

**【核心冲突】**
△太子冷笑："父皇年迈多疑，竟连朕与边关将领的正常书信往来都要过问。"
△心腹跪地进言："殿下，何不借此机会……让那史官'病'上几日？起居注的内容，总得有人把关。"
△太子沉吟片刻："传话给顾长安，就说孤想请他喝杯茶。"

**【爽点释放】**
△顾长安接到邀约时，正悠闲地品着街巷买的馄饨。他不慌不忙地又咬了一口："告诉太子殿下，微臣咳嗽三日，怕过了病气，改日再谢罪。"
△转身便吩咐书童："连夜抄录本月二十三日起居注，送到御史台存档备案。"

**【情感拉扯】**
（内心os：太子拉拢、丞相试探、皇帝猜忌……每逢皇权交接，便是多事之秋。不过无妨，我早有准备。）

**【情感拉扯】**
（内心os：想让我改起居注？可以。但改哪个字、怎么改，得按我的规矩来。）

**【悬念钩子】**
→ 顾长安连夜送去御史台的档案里，究竟藏了什么玄机？
→ 太子茶杯里，泡的是茶还是毒？

## 第4集

**【视觉钩子】**
△三年后。秋风萧瑟，景武帝病重的消息震动朝野。顾长安依旧穿着那身褪色的绯袍，佝偻着腰背立在朝堂角落，执笔的手却稳得可怕。

**【核心冲突】**
△景武帝已无法上朝，只能躺在龙辇中被抬至殿中。他浑浊的双眼扫过跪了一地的大臣，最后定在太子身上："朕……还没死，你们就急着站队了？"
△太子萧珩额头触地，一言不发。朝臣们噤若寒蝉。
△景武帝突然又看向顾长安："顾舍人……起居注记得如何了？"

**【爽点释放】**
△顾长安颤巍巍捧上几卷起居注："回陛下，臣奉命记录，从无遗漏。然臣年事已高，恳请告老还乡。"
△景武帝盯着他看了许久，忽然大笑，笑声里带着苍凉："告老？你才多大年纪……朕还没熬过，你倒想先走？"
△满朝哗然。顾长安面不改色："陛下圣明，臣确已老迈糊涂。"

**【情感拉扯】**
（内心os：圣明？这位陛下登基时才七岁，我亲眼看着他从脓水小儿变成猜忌帝王。三百年了，还没习惯被皇帝叫老吗？）

**【悬念钩子】**
→ 景武帝为何说"朕还没熬过"？难道他看出了什么？
→ 太子即将登基，新朝第一把火，会不会烧向这个"知情太多"的史官？

**【核心冲突】**
△新帝登基第一道旨意不是封赏，而是彻查先帝晚年"妖言案"。
△太监尖声宣旨："着起居舍人顾长安，将先帝最后三年起居注，全部呈送御前！"
△满朝皆惊。顾长安缓缓出列，叩首领旨。

**【爽点释放】**
△退朝后，丞相亲自拦住顾长安："顾大人，太后有话带给您。"话未说完，顾长安便打断他，"太后？哪位太后？先帝可有遗孀？"他故作茫然。
△丞相脸色微变："……先帝驾崩，皇后自然升级为太后。"
△顾长安捋须一笑："哦，老臣糊涂，记性不好，记性不好。"

**【情感拉扯】**
（内心os：先帝驾崩无遗孀，这是我亲眼所见的。至于妖言案……呵，当年那批人，死得可不止一个。）

**【情感拉扯】**
（内心os：新帝要查的不是案，是先帝晚年那些见不得光的事。偏偏那三年起居注里，我用的墨……可是特制的。）

**【悬念钩子】**
→ 特制的墨水记录的内容，如何在关键时刻"消失"？
→ 太后与丞相联手的背后，究竟藏着怎样的交易？

## 第6集

**【视觉钩子】**
△深夜，御书房内灯火通明。年轻的永宁帝萧珩翻阅着起居注，眉头越皱越紧。他猛地将卷宗摔在案上："这三年，父皇召见了谁、说了什么、为何突然性情大变——为何全是空白？"

**【核心冲突】**
△太监跪地禀报："陛下，顾大人说……说年事已高，笔力不济，这几年的记录多有疏漏。"
△永宁帝冷笑："疏漏？朕看他是不想活了。传旨，明日早朝，朕要亲自问问这位'老臣'。"
△殿外，顾长安的马车正缓缓驶离宫门。车帘后，他闭目养神，嘴角却噙着一丝笑意。

**【爽点释放】**
△回到府中，顾长安从暗格中取出一只檀木匣。匣中是三卷用特殊药水书写的档案——真正的起居注。
△他轻声自语："先帝的病、太后的密信、丞相的黑钱……陛下想看？可以。但要看哪一版，就由不得你了。"
△他将檀木匣重新锁好，藏入地下密室。

**【情感拉扯】**

**【情感拉扯】**
（内心os：这三百年来，多少皇帝想灭我的口？最后都化作一抔黄土。而我，依旧是这朝堂上最不起眼的六品小官。）

**【悬念钩子】**
→ 真正的起居注里，究竟记载了怎样的惊天秘密？
→ 明日早朝，顾长安如何应对新帝的雷霆之怒？

**【视觉钩子】**
△翌日早朝，金銮殿气氛压抑得令人窒息。永宁帝高坐龙椅，开门见山："顾长安，朕问你——先帝驾崩前三日，你在何处？"

**【核心冲突】**
△满朝哗然。这是质问，更是一种暗示——先帝之死，另有隐情？
△顾长安颤巍巍出列，叩首道："回陛下，臣那三日均当值记录，不敢有丝毫懈怠。起居注中皆有明载。"
△永宁帝冷笑："既如此，你可知'欺君'之罪如何处置？"

**【爽点释放】**
△顾长安不慌不忙，从袖中取出一卷明黄绸缎："陛下息怒。臣斗胆，将先帝临终遗命录于此处，请陛下御览。"
△永宁帝接过，脸色骤变。遗命上清清楚楚写着：立萧珩为帝，着顾长安保留原职，辅佐新君。
△顾长安叩首："先帝亲口所言，臣一字不落。若陛下有疑，可传当时在场的太医作证——只不过，那几位太医，如今只剩一人尚在了。"

**【情感拉扯】**
（内心os：怕了？你父皇临终前握着我的手说"有劳顾卿"，你可知道？你不知道。你只知道我看了太多、记了太多。）

**【情感拉扯】**
（内心os：但这遗命是真的。太医的证词也是真的。我从不说谎，只是……选择性说真话而已。）

**【悬念钩子】**
→ 唯一幸存的太医，会成为压垮永宁帝的最后一根稻草吗？
→ 顾长安手中，是否还握着更多"先帝遗命"？

## 第8集

**【视觉钩子】**
△一月后，御花园设宴。永宁帝亲自主持，顾长安也在席间。席间推杯换盏之际，永宁帝突然举杯："顾卿，朕敬你三朝元老。"

**【核心冲突】**
△满座皆惊。三朝？先帝、景武帝……还有更早的？
△永宁帝似笑非笑："听闻顾卿历经三朝，这说法可有错？"
△顾长安举杯的手微微一顿，随即恢复如常："陛下说笑。臣不过一介史官，哪敢称元老。"

**【爽点释放】**
△永宁帝步步紧逼："那朕再问你——你见过的高祖皇帝，是什么模样？"
△顾长安轻咳两声："陛下恕罪，臣未曾见过高祖……那都是前朝事了。臣入仕时，高祖已驾崩二十余年。"
△永宁帝盯着他看了半晌，忽然大笑："好，好一个'未曾见过'！"举杯一饮而尽，不再追问。

**【情感拉扯】**
（内心os：他在试探。高祖是本朝开国皇帝，若我说见过，便暴露了长生之谜；若说没见过，便坐实了"三朝元老"的谎言。他在等我露破绽……可惜，我等了三百年，早已滴水不漏。）

**【情感拉扯】**
（内心os：但这皇帝不好糊弄。得想个法子，让他觉得杀我不值。）

**【悬念钩子】**
→ 永宁帝究竟掌握了什么线索？
→ 顾长安心中，那张"以退为进"的底牌是什么？

## 第9集

**【视觉钩子】**
△深夜，顾长安的书房迎来一位不速之客——竟是太后身边的掌事嬷嬷。她捧着一只锦盒，神色复杂："顾大人，太后有话：识时务者为俊杰。"

**【核心冲突】**
△锦盒打开，里面是一杯毒酒与一柄匕首。
△嬷嬷冷冷道："太后说，您知道的太多了。与其日后被陛下清算，不如……自己了断。太后会赐您一个风光的葬礼。"
△顾长安端起毒酒闻了闻，点点头："好酒。只是……太后似乎忘了一件事。"

**【爽点释放】**
△顾长安从袖中取出一封信，轻轻放在桌上："这是太后当年与丞相往来密信的手抄本。原件在御史台存档，副本……臣已托人送往边关藩王处。"
△嬷嬷脸色惨白。顾长安将毒酒推回她手中："劳烦回禀太后：臣老了，糊涂了，什么都不记得。臣只想安度余生，别无他求。"
△嬷嬷踉跄离去。顾长安独坐灯下，轻声自语："又过了一关。"

**【情感拉扯】**
（内心os：太后啊太后，你当年毒杀景武帝扶持你儿子登基，以为神不知鬼不觉？可惜，我亲眼所见，亲笔所记。）

**【情感拉扯】**
（内心os：三百年来，想杀我的人排成排。最后他们去哪了？都去投胎了。而我还在。）

**【悬念钩子】**
→ 太后与丞相互通的密信里，究竟藏着怎样的惊天阴谋？
→ 边关藩王收到副本后，会掀起怎样的风暴？

## 第10集

**【视觉钩子】**
△三年后，边关藩王以"清君侧"之名起兵。永宁帝措手不及，朝中丞相被当场拿下，太后被打入冷宫。顾长安依旧每日点卯，悠然地坐在史馆里抄录起居注。

**【核心冲突】**
△新帝——不，旧帝萧珩被囚禁于宗人府。消息传来时，顾长安正在喝茶。他放下茶盏，淡淡道："可惜了。"
△书童不解："大人可惜什么？"
△顾长安摇头："可惜他不懂。皇位这东西，坐上去是一生，躺下去也是一生。何必呢。"

**【爽点释放】**
△新帝——曾经的藩王登基，第一道旨意便是召见顾长安。年轻的帝王好奇地问："你就是那个三朝老臣？朕的父皇说你是'活历史'，朕倒想问问——朕这江山，能传几代？"
△顾长安捋须一笑："陛下，这话该问史官，还是问天命？老臣只管记录，不敢妄议。"
△帝王大笑："好！朕喜欢你这滑头！官升一级，继续给朕记！"

**【情感拉扯】**
（内心os：又换了一个。三百年来换了四个皇帝，丞相杀了七个，太后赐了十二道毒酒……而我，依然是六品起居舍人——哦，现在是五品了。）

**【情感拉扯】**
（内心os：升官？无所谓。重要的是，这把龙椅又稳了几年。我可以继续苟着，继续看戏，继续……活很久很久。）

**【悬念钩子】**
→ 顾长安的真实身份，究竟还能隐藏多久？
→ 当下一个王朝更迭时，这位"活历史"又将以怎样的姿态，继续他的长生之路？

# 《历史赋我长生，我终苟成万朝元老》

## 第11集

【视觉钩子】
△昏暗的档案库房内，烛火摇曳。满头灰白的顾长安独坐于堆积如山的竹简之间，指尖轻轻拂过一卷泛黄的起居注，封皮上赫然写着"景武帝三十年·春"——正是三十年前太子初立之时。他枯瘦的手微微一顿，嘴角浮现一丝意味深长的弧度。

【核心冲突】
△景和宫内，大皇子萧景珩跪地痛哭，请求父皇收回成命。景武帝却冷冷甩出一封密信：“你自己看！”信中赫然是大皇子与北境军勾结的证据。顾长安执笔立于殿侧，心中暗叹——这位大皇子，终究还是太急了。

【爽点释放】
△就在景武帝即将下旨废黜大皇子之际，顾长安忽然轻咳一声，颤巍巍举起笏板：“陛下，老臣斗胆...起居注中有一事，需当面呈与陛下御览。”景武帝眉头一皱：“讲。”顾长安从袖中取出一卷三十年前的注录，上面清清楚楚记载着——当年力荐立大皇子为太子者，正是当今国丈、太子岳丈王氏家主。景武帝眼中杀意一滞，陷入沉思。

【情感拉扯】
（内心os：呵，想当年王氏跪在我面前求我写下那笔的时候，那眼泪流的...可惜你们王家的账，今日总算能还上一笔了。）
