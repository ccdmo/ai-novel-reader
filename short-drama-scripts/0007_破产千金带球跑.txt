# 穿成破产太子爷的恶毒前女友

---

## 第1集

**【视觉钩子】**
△昏暗潮湿的地下室，斑驳的墙壁渗出水渍，角落里堆着蛇皮袋和塑料桶。床上的女人猛然睁眼——烫着大波浪、浓妆晕成一团、浑身酒气。

**【核心冲突】**
△"承昀，滚回来！"
△手机屏幕显示凌晨3:47。女主阮钰发现自己穿成了睡前看的那本小说里同名同姓的恶毒女配。而就在刚才，原主对着电话那头疯狂输出："穷鬼！废物！当初怎么没被车撞死！"
△按照剧情，三天后她会因为百般羞辱落魄太子爷陆承昀，被扔进养猪场。

**【爽点释放】**
△等等——男主是陆承昀？那个未来会逆袭成京圈真正太子爷、掌控半个科技圈的陆氏掌门人？
△她疯狂回忆小说情节。男主现在确实落魄：公司破产、父亲自杀、名下资产全部冻结。但这只是黎明前的黑暗。三年后，他会东山再起。
△"只要我活过这三天……"阮钰看到一丝生机。

**【情感拉扯】**
→ 她想起书中描写陆承昀被羞辱时的神态——沉默、眼神暗沉如潭水。
→ 现在她就是那个正在疯狂羞辱他的人。
→ 完蛋，这仇恨值已经拉满了。

**【悬念钩子】**
→ 凌晨4点，门锁转动的声音。
→ 男主低沉的声音从玄关传来："……榴莲，我买了。"
→ 他真的去买榴莲了？原主昨晚说想吃，他就连夜去买？

## 第2集

**【视觉钩子】**
△玄关处，男人逆光站着。廉价T恤被汗水浸透，裤脚沾满水泥灰。他手里拎着超市塑料袋，神情淡漠，眼底却有难以掩饰的疲惫。

**【核心冲突】**
△"榴莲？"阮钰脱口而出，声音都在发抖。
△男主陆承昀看了她一眼："你不是要吃？"
△她想起书中这段——原主嫌榴莲太贵又改口要草莓，男主多跑了一趟还被她骂"废物这点事都办不好"。她瞬间求生欲拉满："不用了！草莓……我想吃草莓！"

**【爽点释放】**
△陆承昀的眼神明显变了。
△他盯着她看了三秒，冷淡开口："行。"
△然后转身出门，脚步声消失在楼道里。
△阮钰瘫坐在床上，他什么都没问，但那种审视的目光……像在看一个陌生人。
△他在怀疑她。

**【情感拉扯】**
→ 十分钟后，陆承昀回来了。手里多了盒草莓。
→ 他把草莓放在床头柜上，声音平静："还有别的事吗？"
→ 她摇头。他便转身去卫生间冲澡。
→ 水声哗哗。阮钰盯着那盒草莓，想起书里写他那天淋了雨、发了烧、第二天还坚持去工地……就为了一盒她说想吃又没吃的草莓。
→ 原主是真的恶毒。而她……做不到。

**【悬念钩子】**
→ 卫生间的门虚掩着，她看到镜子映出他后背的伤痕。
→ 那是被人打的。书里没写这些细节。
→ 他这三年……到底经历了什么？

## 第3集

**【视觉钩子】**
△第二天清晨六点，闹钟响起。阮钰睁开眼，发现陆承昀已经不在。破旧桌上压着张纸条："早饭在桌上。别出门。"
△桌上是豆浆和肉包。还热着。

**【核心冲突】**
△她需要出门。她记得剧情——今天下午两点，城西旧货市场有个捡漏机会。一幅被人当赝品卖的画，实际价值六位数。
△但陆承昀让她别出门。
△她纠结再三，还是偷偷跑了出去。
△与此同时，城东工地。
△陆承昀扛着水泥袋，汗水糊了满脸。手机震动，是工头发来的消息：「小陆，下午不用来了，有人找你。」
△他皱眉："谁？"
△工头回复：「说是……你爸以前的熟人。」

**【爽点释放】**
△阮钰在旧货市场淘到了那幅画。六百块，买到真迹。
△她正兴奋地往回走，突然被人拦住去路。
△"阮小姐？"一个穿着得体的中年男人微笑，"陆董想见见您。"
△陆董。书中那个害得陆承昀父亲破产自杀、最后被他亲手送进监狱的人——陆廷山。
△阮钰心跳漏了一拍，面上却冷静下来："陆董？我不认识。"
△男人递上名片："他说，您最近和令公子走得很近……想请您喝杯茶。"

**【情感拉扯】**
→ 她捏着名片的手微微发抖。
→ 书中这个时间段，陆廷山还不知道陆承昀的存在。
→ 是她改变了什么吗？
→ 还是说……有人一直在监视他们？

**【悬念钩子】**
→ 下午五点，地下室门口。陆承昀靠在墙边，神情阴沉。
→ 他看着她一步步走近，声音低沉："去哪了？"
→ 她还没来得及回答，他的目光落在她手里的画轴上。
→ "这是什么？"

## 第4集

**【视觉钩子】**
△狭小的地下室，两人对峙。夕阳从窗户斜照进来，在地上拉出长长的影子。
△陆承昀单手撑在她身侧的墙上，居高临下地盯着她。这个姿势让阮钰后背发凉——他比她高了整整一个头。

**【核心冲突】**
△"问你话。"他的声音很轻，却带着不容拒绝的压迫感，"去哪了？"
△"出……出去逛逛。"她努力保持镇定，把画轴往身后藏。
△他没动，只是微微眯起眼睛："我说过，别出门。"
△"我又不是犯人！"她脱口而出，声音比预想的还要尖锐，"你管得着吗？"
△话一出口她就后悔了。这语气——太像原主了。

**【爽点释放】**
△陆承昀的瞳孔骤然收缩。
△他猛地捏住她的下巴，逼她直视自己："今天谁找你了？"
△阮钰整个人僵住。他知道？他什么都知道？
△"我不知道你在说什么——"
△"少跟我装。"他松开手，冷笑一声，"你以为我不知道有人在查我？从三天前开始，我的工友里多了几个陌生面孔，你昨晚骂我的电话……是不是打给别人听的？"

**【情感拉扯】**
→ 她张了张嘴，却说不出话。
→ 他在试探她。同时，也在暴露自己——他已经察觉到了危险。
→ 书中没写这些。他从什么时候开始怀疑的？
→ 陆廷山。一定是他。
→ "……是你爸。"她轻声说，"陆廷山的人。"

**【悬念钩子】**
→ 陆承昀的表情瞬间凝固。
→ 他盯着她，像是在确认什么。半晌，声音沙哑："你怎么知道这个名字？"
→ 她手心全是汗。她要怎么解释？她总不能说自己是穿书的吧？
→ 这时，门外传来敲门声。一个清朗的男声响起："小钰？在家吗？我是你安柏源哥哥啊——"

## 第5集

**【视觉钩子】**
△门外站着一个阳光帅气的粉发男生。他穿着限量球鞋、戴着墨镜，笑容灿烂得像是从偶像剧里走出来的。
△陆承昀的眼神瞬间冷了下来。

**【核心冲突】**
△"安柏源？"阮钰在记忆里疯狂搜索——书中确实有这么个人，原主的青梅竹马，现在是个小有名气的网红。
△"小钰！你怎么不回我消息！"安柏源一把推开陆承昀，张开双臂要抱她，"想死我了！"
△一只手拦住他的去路。
△陆承昀面无表情："她不方便。"
△安柏源这才正眼看他，目光上下打量，嗤笑："你就是那个……陆家的废物？"他转向阮钰，"小钰，你怎么还跟这种人混在一起？"

**【爽点释放】**
△阮钰脑子里警铃大作。
△原书里安柏源是个炮灰，但陆承昀黑化后第一个收拾的就是他——因为原主在陆承昀落魄时总是拿安柏源刺激他。
△她当机立断，一把挽住陆承昀的手臂："他是我男朋友。有事？"
△空气凝固。
△陆承昀侧头看她，眼底是毫不掩饰的审视。
△安柏源愣住，随即大笑："小钰你开什么玩笑？你之前不是说他连给我们家提鞋都不配——"
△"那是以前。"她打断他，声音冷淡，"现在他是。"

**【情感拉扯】**
→ 她感觉到陆承昀的手臂僵了一下。
→ 手指不自觉收紧，在旁人看不见的角度悄悄捏了捏他的袖子。
→ 求你了，配合一下。
→ 几秒沉默后，一只温热的大手覆上她的手背。
→ 陆承昀看向安柏源，淡淡开口："还有事？"
→ 安柏源脸色变了又变，最后冷笑："行，阮钰，你行。"他甩下一张名片，"想清楚了来找我。"

**【悬念钩子】**
→ 安柏源走后，阮钰松开手就要往里走。
→ "站住。"
△陆承昀的声音从背后传来，带着一丝危险的低沉。
△"谁让你演女朋友的？"
→ 他走近一步，将她抵在门框边，目光灼灼："阮钰，你到底想干什么？"

## 第6集

**【视觉钩子】**
△狭长的走廊里光线昏暗，陆承昀的身影将她完全笼罩。他的呼吸近在咫尺，带着淡淡的汗味和阳光的气息。

**【核心冲突】**
△"我……"阮钰脑子飞速运转，"我就是不想被他烦！"
△"所以拿我挡箭？"他逼近一寸，"你倒是想得挺好。"
△"反正你也不吃亏！"她破罐子破摔，"安柏源那种人缠上来很麻烦的！"
△陆承昀盯着她，忽然嗤笑一声："什么时候怕麻烦了？以前不是最喜欢他那种公子哥围着你转吗？"

**【爽点释放】**
△阮钰心头一跳。他还在试探她。
△她深吸一口气，索性破釜沉舟："人是会变的，陆承昀。"
△"……什么？"
△"我就是变了。"她直视他的眼睛，"怎么，不行吗？"
△他沉默了很久。久到她以为他会发火。
△"行。"他退后一步，声音听不出情绪，"那你最好变彻底点。"
△他转身进了屋，留给她一个意味深长的背影。

**【情感拉扯】**
→ 她愣在原地，心跳得厉害。
→ 他说"变彻底点"是什么意思？是警告，还是……某种认可？
→ 她低头看自己的手，还在微微发抖。
→ 刚才那一瞬间，她好像看到他眼底闪过一丝……困惑？
→ 算了不想了。她先进屋，把那幅画藏好。
→ 六百块变六位数。这是她在这个世界的第一桶金。
→ 有了钱，她才能有更多筹码。

**【悬念钩子】**
→ 深夜，她被噩梦惊醒。
→ 梦里是小说里描写的养猪场，原主被几个人按住，塞进猪圈……
→ 她猛地坐起来，发现陆承昀不在床上。
→ 阳台边，男人背对着她，指尖夹着一支没点燃的烟。
→ 月光下，他的背影显得格外孤独。

## 第7集

**【视觉钩子】**
△凌晨三点半的阳台。陆承昀站在栏杆边，望着远处城市的星星灯火。烟燃了一半，灰烬落在地上。

**【核心冲突】**
△阮钰披着外套走过去，在他身边站定。
△"睡不着？"她问。
△陆承昀没回答，只是侧头看了她一眼："回去睡。"
△"睡不着。"她在他旁边蹲下来，托着腮看他，"你也是。"
△他沉默了一会儿，忽然开口："你变了。"
△又是这句。她心里一紧，等待他的下文。
△"以前的你不会半夜爬起来陪我吹冷风。"他的声音很轻，"也不会……说我是你男朋友。"

**【爽点释放】**
△阮钰决定冒一次险。
△"陆承昀，我问你个事。"
△"说。"
△"如果……我是说如果——"她斟酌着措辞，"有一天你重新站起来，有钱有势了，你会怎么对我？"
△他转过头，认真地看着她："你想我怎么对你？"
△她避开他的目光，声音低了下去："就是问问。"
△"以前的你，"他一字一顿，"我会让她生不如死。"
△她浑身一僵。
△"但现在……"他的目光落在她脸上，带着某种说不清的意味，"我不知道。"

**【情感拉扯】**
→ 她心跳漏了半拍。
→ 书里写陆承昀对原主的恨深入骨髓，甚至在她死后都没有半分动容。
→ 但现在他说"不知道"。
→ 他在动摇。
→ "为什么……会变？"她轻声问。
→ 他没回答，只是把没抽完的烟掐灭，转身往屋里走。
→ 走到门口时，他的声音飘过来："你不用变回以前那样。我……"
→ 他顿了顿，声音低得几乎听不见："不习惯。"

**【悬念钩子】**
→ 第二天一早，阮钰的手机响了。
→ 是条短信，来自未知号码：「阮小姐，我儿子的画在你那里吧？——陆廷山」
→ 她的血液瞬间冰凉。
→ 他怎么知道的？画是昨晚才买的……
→ 除非，从一开始，就有人在监视他们的一举一动。

## 第8集

**【视觉钩子】**
△画室出租屋。阮钰盯着手机屏幕，手指冰凉。陆廷山怎么会知道那幅画？

**【核心冲突】**
△"在看什么？"
△陆承昀的声音从身后传来。他今天休息，难得在家，正坐在那张破沙发上翻一本商业杂志。
△她下意识锁屏："没、没什么。"
△"阮钰。"他放下杂志，"我说过，你不用在我面前装。"
△她抿了抿唇，把手机递给他。
△看完短信，陆承昀的眼神沉了下来。
△"他还真是阴魂不散。"他的声音冷得像淬了冰。

**【爽点释放】**
△"那幅画……"阮钰小心翼翼地开口，"是我在旧货市场买的。不是偷的。"
△"我知道。"
△"但他怎么会知道？"
△陆承昀沉默片刻，忽然问："你什么时候开始对画感兴趣了？"
△她一愣。原主是艺术生，但他显然不太清楚她的专业。
△"我是美院的。"她解释，"学油画的。"
△他挑了挑眉，似乎有些意外。
△"六百块买的，"她继续说，"如果我没看走眼的话，应该是清代的真迹，至少值这个数。"
△陆承昀看她的眼神变了。
△"你怎么知道是清代？"
△她心知失言，但话已出口只能硬着头皮："……直觉。"

**【情感拉扯】**
→ 他没追问，但那双眼睛里的审视更浓了。
→ 阮钰赶紧转移话题："陆廷山到底想干什么？针对你，还是针对那幅画？"
→ "都不是。"陆承昀站起身，走到窗边，"他在试探我。"
→ "试探什么？"
→ "试探我还有没有利用价值。"他冷笑，"我爸死的时候，他吞了陆氏全部资产。但他不知道……我在海外藏了些东西。"
→ 阮钰瞪大眼睛。书里没写这个！
→ "什么东西？"
→ 陆承昀转头看她，目光锐利："你很想知道？"
→ 她连忙摇头："我就随便问问！"
→ 他盯着她看了几秒，忽然笑了。
→ 这是她穿越以来第一次见他笑。不是冷笑，是真正的、带着一丝无奈的笑。
→ "阮钰，你胆子真大。"

**【悬念钩子】**
→ 手机再次震动。
→ 是安柏源发来的消息：「小钰，昨晚的事我道歉。我这有份工作介绍给你，薪资丰厚。需要的话今晚来我工作室谈。地址发你。」
→ 紧接着又一条：「别告诉那个穷小子。」
→ 阮钰还没来得及反应，陆承昀的声音从背后飘来："谁的消息？"
→ "……没人。"
→ 他看了眼她手机的方向，眼神暗了暗。
→ "撒谎的时候，眼睛别乱飘。"

## 第9集

**【视觉钩子】**
△安柏源的工作室在市中心高档写字楼里。落地窗俯瞰全城，装修奢华。
△阮钰站在门口，深吸一口气，还是推门进去了。

**【核心冲突】**
△"小钰！"安柏源热情地迎上来，"我就知道你还是想通的。"
