# AI Novel Reader

AI 小说阅读器 - 简洁现代的在线阅读体验。

🌐 在线访问：https://ccdmo.github.io/ai-novel-reader/

## 功能特性

- 📚 小说列表展示
- 📖 章节导航阅读
- 📥 TXT / EPUB 下载支持
- 📱 响应式设计，适配移动端

## 添加新小说

在 `novels/` 目录下创建新的小说文件夹，例如 `novels/my-novel/`，在该文件夹中放入 `index.html` 即可。

## 部署说明

本项目通过 GitHub Actions 自动部署到 GitHub Pages。每次推送到 `main` 分支都会自动触发构建和发布。
